[Remix: A Newly Open Sourced React-Based Framework for Web Apps](https://remix.run/) — Remix has been around for [over a year](https://dev.to/dabit3/a-first-look-at-remix-run-449a) as ‘sponsorware’ but after picking up a *lot* of ardent supporters (and [$3m of seed funding](https://remix.run/blog/seed-funding-for-remix)) the project has now been opened up. Bringing modern Web tech, tooling, and server-side rendering concepts together, there are a few things to check out if you're keen to learn more:

- The project's official [▶️ 50 second Apple-esque commercial/pitch](https://www.youtube.com/watch?v=9FDV-VCq5XE).
- [▶️ Kent C Dodd's multi-hour tutorial walkthrough](https://www.youtube.com/watch?v=hsIWJpuxNj0). Almost a 'course' in scope.
- [An introduction to building a Remix app](https://flaviocopes.com/remix/) with Flavio Copes.
- A [▶️ seven minute review](https://www.youtube.com/watch?v=A9frCHnOeHo) with Ahmad Awais.
- Want to play? You can get building a Remix app right away [thanks to StackBlitz's WebContainers](https://blog.stackblitz.com/posts/remix-runs-on-webcontainers/) at [remix.new](https://remix.new/).
