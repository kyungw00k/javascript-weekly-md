🛠 Code & Tools

[Deno 1.9 Released](https://deno.com/blog/v1.9) — Node's anagrammed cousin continues to get even better. 1.9 brings a [native HTTP/2-compliant Web server](https://deno.com/blog/v1.9#native-http%2F2-web-server) (which seems to beat Node's in basic benchmarks), support for `blob` and `data` URLs, faster interop with the underlying Rust code, and more. *"Why use Deno,"* you ask? Think V8, but with the layer on top built in Rust, with a focus on extra security, first class TypeScript support, and offering browser-compatible APIs wherever possible.

The Deno Team

[Announcing `react-pdf` v2.0](https://react-pdf.org/blog/announcing-react-pdf-v2) — While it remains backward-compatible with v1, the new version of react-pdf is a *complete reimplementation* from scratch.
