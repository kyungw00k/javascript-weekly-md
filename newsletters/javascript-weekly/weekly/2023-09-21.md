# JavaScript Weekly Issue 655: September 21, 2023

## Top Stories

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/kzmqkvwcnarz0io8kxot.jpg)

▶ [TypeScript Origins: The Documentary](https://www.youtube.com/watch?v=U6s2pdxebSo) — You know you’ve made it when you get your own documentary! This has *just* dropped but is well produced, packed with stories from TypeScript’s co-creators, users, and other folks at Microsoft, and kept me entertained. It goes particularly deep into the motivations and process behind TypeScript's creation, including why Microsoft felt it was worth pursuing at all.

OfferZen Origins

🔥 The takes get spicer 25+ minutes in as various TypeScript users chip in with their opinions, and even Daniel Rosenwasser, now program manager of the TypeScript team, says he initially worried that Microsoft might *"f**k it all up."* 😅 Ryan Dahl pops up about an hour in too.. so I hope you've got some time spare.

[Patterns for Reactivity with Modern Vanilla JavaScript](https://frontendmasters.com/blog/vanilla-javascript-reactivity/) — When data changes, you often want to do things, and the process that makes that happen is *reactivity.* While many libraries or frameworks like React or Vue offer reactive solutions out of the box, you can do it all with vanilla JavaScript too, and this post digs into many of the patterns you might need to use, complete with examples.

Marc Grabanski

![](https://copm.s3.amazonaws.com/04865a0b.png)

[Handsontable: Data Grid With Spreadsheet Superpowers](https://handsontable.com/demo) — There are a lot of JavaScript data grids out there, but none are quite like Handsontable. Its Excel-like design, keyboard shortcuts, and navigation make it the perfect tool for developers.

Handsontable sponsor

[Deno 1.37: Modern JavaScript in Jupyter Notebooks](https://deno.com/blog/v1.37) — Not content to let Bun take all the headlines, Deno has come up with something pretty neat here. Deno 1.37 ships with [a Jupyter Notebook integration](https://deno.com/blog/v1.37#jupyter-notebook-integration) so you can create interactive REPL sessions but using the JavaScript you know and love, rather than Python.

The Deno Team

[JavaScript Minification Benchmarks](https://github.com/privatenumber/minification-benchmarks) — A frequently updated benchmark suite and results comparing the speed and quality of JavaScript minification across a variety of tools including esbuild, Babel, Bun, SWC, and Uglify.

Hiroki Osame

**⚡️ IN BRIEF:**

- The Svelte team offers a sneak peek at the forthcoming Svelte 5 and [introduces the idea of 'runes'](https://svelte.dev/blog/runes): *"Like every other framework, we've come to the realisation that [Knockout](https://knockoutjs.com/) was right all along."*
- [Happy 27th birthday to JScript!](https://www.webdesignmuseum.org/web-design-history/jscript-1-0-1996) JScript was a version of JavaScript that Microsoft put into IE 3.0 primarily to avoid trademark issues around the name of JavaScript. No-one cares much about that anymore, despite [Oracle still technically holding](https://tinyclouds.org/trademark) the *JavaScript* trademark.
- Folks are discussing whether [Node.js needs a mascot](https://github.com/nodejs/admin/issues/828) of its own.
- Want to support anyone doing great work on JavaScript projects? You can [nominate them for a GitHub Star.](https://stars.github.com/nominate/)
- If you rely upon the npm package download counts at all, the daily counts for all packages was *zero* on September 13 and 14 ([example](https://api.npmjs.org/downloads/range/2023-09-10:2023-09-21/react)). Why? *"This issue remains under investigation,"* says npm support.
