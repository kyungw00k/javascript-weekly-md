# JavaScript Weekly Issue 675: February 15, 2024

## Top Stories

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/ahsccrqo1gkdlhzwc0uv.jpg)

[Tempo: An Easier Way to Work with Dates](https://tempo.formkit.com/) — We think they nailed their own summary: *“A collection of utilities for working with the native `Date` object — an important distinction from other libraries that provide custom date primitives. Under the hood, Tempo mines `Intl.DateTimeFormat` to extract complex data like timezones offsets and locale aware date formats, giving you a simple API to format, parse, and manipulate dates.”* Fantastic docs, too.

FormKit

[LLRT (Low Latency Runtime): Amazon's New JS Runtime](https://github.com/awslabs/llrt) — Amazon has open sourced a runtime entirely focused on fast startups – something that’s useful for ephemeral use cases like serverless (**cough* AWS Lambda*). It uses Fabrice Bellard’s [QuickJS](https://bellard.org/quickjs/) as the underlying engine, so has almost full ES2023 spec support out of the box.

Amazon Web Services Labs

**IN BRIEF:**

- You know how features and proposals for the ECMAScript language [pass through a multi-stage process](https://tc39.es/process-document/) with the TC39 committee? **There's a new stage: stage 2.7!** The new stage is [what 3 used to be](https://twitter.com/robpalmer2/status/1755362667416748266) with 3 now having extra testing requirements.
- Talking of TC39, here's [the latest state of different proposals](https://dev.to/hemanth/updates-from-the-100th-tc39-meeting-4j2f) covered at the recent 100th TC39 meeting.
- There have been [some updates on the state of Express](https://github.com/expressjs/discussions/issues/160) and suggested plans for Express 5.0, 6.0 and 7.0.
- [Google has unveiled](https://blog.google/technology/ai/google-gemini-next-generation-model-february-2024/) the newest version of its *Gemini* AI model. Of particular note is its support for 1 million tokens of context, as shown in [▶️ this example working upon the entirety of three.js's source code](https://www.youtube.com/watch?v=SSnsmqIj1MI) (~820k tokens).
- A new JavaScript registry is on the way called [jsr.](https://jsr.io/waitlist) Waitlist only, for now.
