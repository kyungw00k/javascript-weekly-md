Quick Bits

- [SpiderMonkey](https://spidermonkey.dev/) is Mozilla's long standing JavaScript engine as used in Firefox and other projects. It now has an [all new homepage](https://spidermonkey.dev/) and.. drum roll.. a logo for the first time ever.
- Ben Beattie-Hood has released [Modern TypeScript](https://leanpub.com/moderntypescript), an e-book aiming to raise money for the Australian Red Cross. This [18-minute video](https://www.youtube.com/watch?v=v86ggw3LoW0) covers what's in the book.
- [V8 9.1](https://v8.dev/blog/v8-release-91) is on the way. It's one of the less feature packed releases but adds top level `await` by default (already the case in some users of V8, such as Blink), private brands check syntax by default, and some perf improvements.
- Up and coming toolchain [Rome](https://rome.tools/) has [formed a company, raised some money, hired folks](https://rome.tools/blog/announcing-rome-tools-inc/).
- A look at [what's new in RxJS 7](https://medium.com/volosoft/whats-new-in-rxjs-7-a11cc564c6c0), the reactive extensions for JavaScript.
