[asciinema-player, Now 4x Smaller and 50x Faster with JS *and* Rust](https://blog.asciinema.org/post/smaller-faster/) — [asciinema](https://github.com/asciinema/asciinema) is a Python-based terminal screen recorder (think *Camtasia* but for text!) but to play its recordings on the Web [asciinema-player](https://github.com/asciinema/asciinema-player) is used. Originally written in ClojureScript, a mixed JS + Rust build (plus a little [SolidJS](https://www.solidjs.com/)) has *hugely* improved its performance (the demo on the page is neat to play). Will we continue to see more collaborations between Rust and JavaScript in future? Definitely.

Marcin Kulik

[Mitosis: Write Components Once, Run Everywhere](https://github.com/BuilderIO/mitosis#readme) — In simple scientific terms, mitosis is a process where a cell divides into two child cells, but in this case it’s a single component code base that can compile into different forms usable with Vue, React, Solid, Angular, Svelte, and more. The [playground](https://mitosis.builder.io/?outputTab=vue) lets you see how it works out.

Builder.io

Enter the [JSON modules proposal](https://github.com/tc39/proposal-json-modules) which is at stage 3 with experimental support available in Node or within the Chrome 91 browser.
