## 🥳 And a bit of fun..

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/iyju9nok5xcuksg4e75z.jpg)

[p5.js](https://p5js.org/) is a popular ‘creative coding’ library that takes a lot of inspiration from [Processing](https://processing.org/). p5 makes it easy to create digital art and interactive experiences without a lot of boilerplate (check out how simple [this 'smoke particles' demo](https://p5js.org/examples/math-and-physics-smoke-particle-system/) is, for example).

[p5.js v1.10.0](https://github.com/processing/p5.js/releases/tag/v1.10.0) has just landed, but the reason I wanted to mention it is because of how much fun I've had throwing AI generated code at [the online p5 editor](https://editor.p5js.org/), and if you have access to tools like ChatGPT, you can do it too.

For example, give ChatGPT a prompt like:

```
Write some p5.js code that renders the JavaScript "JS" logo.
```

Copy and paste the result into [the editor](https://editor.p5js.org/), and:

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/m1z8chddqxmolpvzaara.jpg)

It's not perfect, but it's pretty good for a first try (and easy to 'fix' by hand).

Different LLMs take totally different approaches. Consider what Anthropic's Claude 3.5 gave me, for example:

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/z5lx14s4yia3cccne0xg.jpg)

Eeek..

Well, anyway, there's a lot of fun to be had, and you can prompt LLMs to improve the results or customize things as you wish. Have some fun with it, it's pretty cool what you can produce.
