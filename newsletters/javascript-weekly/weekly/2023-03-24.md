## Top Stories

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/kbzwsvo1eytwa90htzw8.jpg)

[Speeding Up the JavaScript Ecosystem: npm Scripts](https://marvinh.dev/blog/speeding-up-javascript-ecosystem-part-4/) — The latest in what has been a fascinating series on finding ‘low hanging fruit’ when it comes to performance in the JavaScript world. The author explains it best himself:

> *“‘npm scripts’ are executed by JavaScript developers … all the time. Despite their high usage they are not particularly well optimized and add about 400ms of overhead. In this article we were able to bring that down to ~22ms.”*

Marvin Hagemeister

[Playwright v1.32 – Now with UI Mode](https://github.com/microsoft/playwright/releases/tag/v1.32.0) — The popular Web testing and automation framework is taking more steps toward ground currently served by tools like Cypress by offering a ‘UI mode’ that lets you explore, run and debug tests in a UI environment, complete with watch mode. [▶️ This video](https://www.youtube.com/watch?v=jF0yA-JLQW0) provides a good introduction.

**IN BRIEF:**

- GitHub [had to update its RSA SSH host key today](https://github.blog/2023-03-23-we-updated-our-rsa-ssh-host-key/) so you may see security related warnings when pushing and cloning. It's easy to fix, but check the new fingerprint [matches](https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/githubs-ssh-key-fingerprints) – it's for your own security.
- *The New Stack* caught up with Svelte's Rich Harris [on SvelteKit and what's coming for Svelte 4](https://thenewstack.io/rich-harris-talks-sveltekit-and-whats-next-for-svelte/).
- The React team shared some [cutting edge updates on what they're working on](https://react.dev/blog/2023/03/22/react-labs-what-we-have-been-working-on-march-2023) including React Server Components and an optimizing compiler.
- If you were experiencing errors on the official Node site last week, [here's the (detailed) post mortem of why](https://nodejs.org/en/blog/announcements/node-js-march-17-incident). Config errors and inappropriate caching, mostly.
- ✨ Did you know there's a market in fake GitHub stars? Some developers [analyzed some repos to learn more about it](https://dagster.io/blog/fake-stars).
