## 🛠 Code & Tools

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/txk1w8kctbkc8tpt3y1d.jpg)

[Croner 7.0: Cron for JavaScript and TypeScript](https://github.com/hexagon/croner) — Trigger functions upon the schedule of your choice using the classic [cron syntax.](https://en.wikipedia.org/wiki/Cron#CRON_expression) Works in Node, Deno, Bun *and* the browser, across time zones, offers error handling and overrun protection, and more. There’s an [live demo](https://jsfiddle.net/hexag0n/hoa8kwsb/) on JSFiddle. [v7.0](https://github.com/Hexagon/croner/releases/tag/7.0.0) adds support for specifying the *n-*th weekday.

Hexagon

[Linkify 4.1: Link Up URLs, Email Addresses, and More in Plain Text](https://linkify.js.org/) — Given some plain text containing things like links and emails, Linkify will generate the correct code to display it on the Web. Not just URLs or emails, but hashtags, @mentions, IP addresses, and more – see the homepage for examples. [GitHub repo](https://github.com/Hypercontext/linkifyjs).
