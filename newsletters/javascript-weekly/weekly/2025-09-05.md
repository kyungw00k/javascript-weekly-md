[Mediabunny: A Complete Media Toolkit for JavaScript](https://mediabunny.dev/) — Supporting both browsers and Node.js, this library lets you read, write and convert popular media file formats (e.g. MP4, MP3, [and more](https://mediabunny.dev/guide/supported-formats-and-codecs)) without needing to lean on dependencies like FFmpeg. You can [make thumbnails](https://mediabunny.dev/examples/thumbnail-generation/), [extract metadata](https://mediabunny.dev/examples/metadata-extraction/), [write code that gets converted into a video](https://mediabunny.dev/examples/procedural-generation/), and more. [GitHub repo.](https://github.com/Vanilagy/mediabunny)

Vanilagy

[Why Do Browsers Throttle JavaScript Timers?](https://nolanlawson.com/2025/08/31/why-do-browsers-throttle-javascript-timers/) — Did you know the zero in `setTimeout(0)` doesn’t really mean *zero*? Browsers clamp timers to at least a few milliseconds or more, slowing down code that relies on them. Luckily, there are alternative options, which Nolan demonstrates and benchmarks.

Nolan Lawson

## 🛠 Code & Tools

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/csbiyt4i3hxzhyqbrrjg.jpg)

[Peaks.js 4.0: UI Component for Interacting with Audio Waveforms](https://codeberg.org/chrisn/peaks.js) — A project originally spawned from the BBC’s R&D department, Peaks renders audio waveforms to a canvas element and allows scrolling, zooming, and the sort of things you might otherwise see in an audio editor. *LGPL licensed.*

Chris Needham et al.

💡 There are other options in this space too, including the newly released [Waveform Renderer 1.0](https://waveform-renderer.vercel.app/) (complete with [interactive demo](https://waveform-renderer.vercel.app/demo/)) which is entirely focused on waveform rendering, as well as [Wavesurfer.js.](https://wavesurfer.xyz/)

[Redux Toolkit 2.9: The 'Batteries-Included' Toolset for Redux](https://github.com/reduxjs/redux-toolkit/releases/tag/v2.9.0) — The long-standing toolset for working with Redux for state management gets a performance update with a rewrite of RTK Query’s subscription and polling systems, automatic aborting of in-flight requests when a cache entry is removed, and more.

Mark Erikson

[react-window 2.0: Render Large Lists of Data Quickly](https://react-window.vercel.app/) — A component library for rendering large lists of data quickly and without the performance problems that often go along with that. Brian Vaughn
