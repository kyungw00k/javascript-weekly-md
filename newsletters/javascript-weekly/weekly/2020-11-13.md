## 👀 A Correction

[The File System Access API: Simplifying Access to Local Files](https://web.dev/file-system-access/) — Several issues ago we mistakenly referred to [this API’s spec](https://wicg.github.io/file-system-access/) as an ‘open standard’ when it's just a spec. It's Chrome only (for now), not a *W3C standard*, though it remains an interesting idea. *(Thanks to reader Šime Vidas for [noting](https://webplatform.news/issues/2020-11-06) our mistake and noting that the path from the WICG to a W3C standard is a long one indeed!)*
