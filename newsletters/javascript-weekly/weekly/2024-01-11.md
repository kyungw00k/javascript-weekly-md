There are a few demos, but the [simplest one](https://gildas-lormeau.github.io/zip.js/demos/demo-create-file.html) sold me on its own. [GitHub repo](https://github.com/gildas-lormeau/zip.js).

[Element Plus: Vue 3 Component Library](https://element-plus.org/en-US/) — Out of the box you get a large variety of clean components with a minimal vibe. You can [see them in action here](https://element-plus.org/en-US/component/button.html).

Element Plus Team

[Add Authorization, MFA, Biometrics and More to Your JavaScript App in Just Minutes](https://fusionauth.io/ebooks/modern-guide-to-oauth?utm_source=jsweekly&utm_medium=paid_other&utm_campaign=jan24_newsletter&utm_content=modern_guide_oauth) — It's about time that somebody talked some sense about OAuth and JavaScript. So we did. You're welcome.

FusionAuth sponsor

[RE2JS: Linear Time Matching for Regular Expressions](https://github.com/le0pard/re2js) — RE2 is a regex engine built by Google designed to operate in time proportional to the size of the input, in order to avoid so-called ‘ReDoS’ problems backtracking can cause. There are [bindings for Node.js](https://github.com/uhop/node-re2), but this version works in the browser.

Oleksii Vasyliev

[Penrose: Beautiful Math Diagrams by Writing Plain Text](https://penrose.cs.cmu.edu/) — It might sound a little like [Mermaid](https://mermaid.js.org/) (indeed, their homepages are strikingly similar) but its scope feels somewhat more math-focused – there are [many examples](https://penrose.cs.cmu.edu/examples).

Penrose Contributors

[DOMPurify 3.0.8: A Fast, Tolerant XSS Sanitizer for HTML](https://github.com/cure53/DOMPurify) — Supports all modern browsers and is heavily tested. There’s [a live demo here](https://cure53.de/purify).

Cure53

[remoteStorage: Cross Browser/Device Persistence](https://github.com/FrigadeHQ/remote-storage) — Combines the `localStorage` API with a remote server to persist data across browsers and devices – so basically, imagine `localStorage` but remote.
