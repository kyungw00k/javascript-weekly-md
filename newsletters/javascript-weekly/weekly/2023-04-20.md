## Top Stories

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/ianh3bgg48t6a4lowziv.jpg)

[Introducing npm Package Provenance](https://github.blog/2023-04-19-introducing-npm-package-provenance/) — As stewards of the official npm registry, GitHub has been keen to improve its security. As of this week, if you build npm packages through GitHub Actions, you can publish the *provenance* of a package which gives users a way to verify which repo a package was built from *and how.* The folks at Socket took [a closer look at how it works.](https://socket.dev/blog/npm-provenance)

DeHamer and Harrison (GitHub)

[What's New in ECMAScript 2023, Briefly](https://pawelgrzybek.com/whats-new-in-ecmascript-2023/) — Last week we mentioned the progression of the [ECMAScript 2023 spec](https://tc39.es/ecma262/2023/) but it’s hardly bedtime reading. This post looks more briefly at a few of the new additions including `Array#findLast`, hashbangs, and `Array#toSorted`.

Pawel Grzybek

[New Course by Kyle Simpson: Practical Problem Solving with Algorithms](https://frontendmasters.com/courses/algorithms-practice/?utm_source=javascriptweekly&utm_medium=email&utm_content=algorithmspractice) — Walk through four fun challenges with Kyle in this new video course. You'll learn how to apply algorithms with techniques like recursion, traversals, and acyclic paths, cover memoization to avoid garbage collection, and come away writing more performant code.

[Node.js v20 (Current) Released](https://nodejs.org/en/blog/announcements/v20-release-announce) — Released just two days ago, the newest major version of Node.js includes a variety of interesting updates, including:

- **An experimental permissions model.** This is a feature [Deno has at its core](https://deno.land/manual@v1.32.4/basics/permissions) but now you can restrict access to certain capabilities in Node too.
- **V8 11.3** (which includes [Regexp `/v` flag support](https://chromestatus.com/feature/5144156542861312))
- **The `node:test` test runner goes stable.**
- Official support for Windows on ARM64.
- Improvements to the single executable packaging mechanism.

The Node.js Team

[As per policy,](https://nodejs.dev/en/about/releases/) Node v20 will become the main LTS release this October.

[Vite 4.3: Now It's Even *'Faaaaster'*](https://vitejs.dev/blog/announcing-vite4-3.html) — A minor release of the popular frontend toolchain, but one that focuses on *performance.* The creatively titled [How we made Vite 4.3 faaaaster](https://sun0day.github.io/blog/vite/why-vite4_3-is-faster.html) digs into the details.
