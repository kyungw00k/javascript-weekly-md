[Feature Watch: ECMAScript 2021](https://2ality.com/2020/09/ecmascript-2021.html) — Dr. Axel always keeps tabs on the new features coming to future versions of ECMAScript and he’s now started to do the same for the forthcoming ES2021. Related to this TC39 has advanced [a number of proposals this week](https://twitter.com/robpalmer2/status/1308292088484696066) (though they're not all guaranteed for ES2021 by any means):

- [Class static initialization blocks](https://github.com/tc39/proposal-class-static-block) to stage 2
- [`.item()` for indexables](https://github.com/tc39/proposal-item-method) to stage 3 ([more on this by Dr. Axel.](https://2ality.com/2020/09/item-method.html))
- [Import Assertions](https://github.com/tc39/proposal-import-assertions) to stage 3
- [Intl.DisplayNames](https://github.com/tc39/proposal-intl-displaynames) to stage 4
- [Intl.Enumeration](https://github.com/tc39/proposal-intl-enumeration) to stage 2
- and [a few others..](https://dev.to/hemanth/updates-from-78th-meeting-of-tc39-2cbm)

Dr. Axel Rauschmayer

[billboard.js 2.1.0: A D3-Based Chart Library](https://medium.com/@alberto.park/billboard-js-2-1-0-release-another-big-step-for-performance-improvement-more-9f0b9357d9d5) — Supporting over 10 different chart types or a combination of such, Billboard is a powerful option for rendering attractive charts – check out [the demos here.](https://naver.github.io/billboard.js/demo/) v2.1 increases performance for larger datasets both during rendering and resizing – there’s a new theme available too.

[V8 Release 8.6](https://v8.dev/blog/v8-release-86) — The latest version of the V8 JavaScript engine is now in beta until its eventual release with Chrome 86 (and, eventually, a future version of Node). The codebase is now [more respectful](https://v8.dev/docs/respectful-code), there are speedups in `Number.prototype.toString`, and a JavaScript fuzzer called [js-fuzzer](https://chromium-review.googlesource.com/c/v8/v8/+/2320330) has been open sourced.

Ingvar Stepanyan (V8 Team)

**⚡️ Quick bytes:**
