# JavaScript Weekly Issue 684: April 18, 2024

## Top Stories

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/witsoytxgchsvp3nmrpp.jpg)

[Quill 2.0: A Powerful Rich Text Editor for the Web](https://quilljs.com/) — A major release and significant modernization for the open source WYSIWYG editor. In [Announcing Quill 2.0](https://slab.com/blog/announcing-quill-2-0/), we learn about Quill’s transition to TypeScript and improved use of modern browser features, but there’s [more going on](https://github.com/quilljs/quill/releases/tag/v2.0.0) too, such as its ESM packaging. Want to play with some code? [There’s a playground.](https://quilljs.com/playground/snow)

Slab Inc.

[Airbnb's Extensive JavaScript Style Guide](https://github.com/airbnb/javascript) — It’s been years since we mentioned this popular, opinionated style guide, but it keeps getting little tweaks and repairs and remains a handy resource nonetheless.

Airbnb

![](https://copm.s3.amazonaws.com/742f992f.png)

[WorkOS: Enterprise-Grade Auth for Modern SaaS Apps](https://workos.com/?utm_source=cpjavascript&utm_medium=newsletter&utm_campaign=q22024) — WorkOS supports both the foundational auth and complex enterprise features like SSO. It provides flexible and easy-to-use APIs, helping companies like Vercel, Loom, and Webflow become Enterprise Ready. Best of all, WorkOS User Management supports up to 1 million MAUs for free.

WorkOS sponsor

[Upgrading jQuery: Working Towards a Healthy Web](https://blog.jquery.com/2024/04/17/upgrading-jquery-working-towards-a-healthy-web/) — jQuery remains all over the Web, and the jQuery team and OpenJS Foundation have joined forces to make sure sites get up to date. Their ['Healthy Web Checkup' tool](https://healthyweb.org/) can tell you if the version of jQuery on a site is outdated (.. most likely 'yes' 😅).

Timmy Willison (jQuery)

[Biome v1.7: Faster Formatting and Linting, Now Even Easier to Migrate To](https://biomejs.dev/blog/biome-v1-7/) — [Biome](https://biomejs.dev/) is an increasingly compelling, all-in-one JavaScript, TypeScript and JSX Prettier-compatible formatter and linter. v1.7 makes it easier to migrate to from ESLint and Prettier, can emit machine-readable JSON reports, and has some rule updates.

Biome Core Team

**IN BRIEF:**

- 🤡 [▶️ Interview with Senior JS Developer 2024](https://www.youtube.com/watch?v=aWfYxg-Ypm4) is a satirical(?) take on the state of modern development and increasingly complex ecosystems. Watch to either laugh or cry. *(6 minutes.)*

- [f(x) (a.k.a. fx)](https://fx.wtf/) is a great CLI tool for viewing JSON data. [It now includes](https://github.com/antonmedv/fx/releases/tag/34.0.0) a JS engine so you can process JSON with JavaScript one-liners at the terminal, too.
- [Joyee Cheung shares a trip report](https://nodejs.org/en/blog/events/collab-summit-2024-london) from Node.js's latest collaboration summit, touching on the topics the Node team is thinking about lately.
- The folks at Deno [go into depth about how they built JSR](https://deno.com/blog/how-we-built-jsr), the newest JavaScript registry, and how it can operate at scale.
- ["Much faster JSX element creation"](https://github.com/facebook/react/pull/28768) is on the way for React developers.
