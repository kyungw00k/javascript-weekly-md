**⚡️ IN BRIEF:**

- 🫢 [Svelte is converting from TypeScript to JSDoc](https://github.com/sveltejs/svelte/pull/8569) ([example](https://github.com/sveltejs/svelte/pull/8569/commits/ea4e7931e41b4606abec51c8a67e1e3c282b8de8)).. sort of. Rich Harris popped up on Hacker News to provide [some all important context](https://news.ycombinator.com/item?id=35892250) but the ultimate result will be smaller package sizes and a better experience for Svelte's maintainers.
- React now has [official 'canary' releases](https://react.dev/blog/2023/05/03/react-canaries) if you want to use newer features than in the stable releases but still be on an officially supported channel.
- Newly released [Firefox 113](https://www.mozilla.org/en-US/firefox/113.0/releasenotes/) lets you override JS files in its debugger.
- No stranger to controversy, Ruby on Rails's David Heinemeier Hansson (DHH) tweeted: *🐦 ["TypeScript sucked out much of the joy I had writing JavaScript."](https://twitter.com/dhh/status/1655076668787097607)*
