**IN BRIEF:**

- [Node.js v20.19.0 (LTS)](https://nodejs.org/en/blog/release/v20.19.0) is a notable release for a maintenance branch of Node as an exception has been made to the maintenance policy for backporting `require(esm)` support, which is now enabled by default.
- The latest version of Chrome (and preview versions of Firefox and Safari) support [two new HTML attributes, `command` and `commandfor`](https://developer.chrome.com/blog/command-and-commandfor), for adding declarative actions to HTML buttons without directly requiring JavaScript.
