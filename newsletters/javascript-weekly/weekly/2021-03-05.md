# JavaScript Weekly Issue 528: March 5, 2021

## Top Stories

😆 If this were an episode of *Friends*, it would be called something like *"The One With All The Big Releases"* — enjoy! __ *Peter Cooper, editor*

[jQuery 3.6.0 Released](http://blog.jquery.com/2021/03/02/jquery-3-6-0-released/) — I’m always delighted to share jQuery news for a few reasons. First, it’s one of the best JavaScript nostalgia trips we still have. Second, jQuery is still *so heavily used* whether in CMSes or even by developers who simply continue to appreciate it. The updates are minor here, but we’re promised a jQuery 4.0 is on the way.

Timmy Willison

[V8 (and Chrome) to Get Faster Release Cycle](https://v8.dev/blog/faster-releases) — Google Chrome is [speeding up its release cycle](https://developer.chrome.com/blog/faster-release-cycle/) from six weeks to four weeks (so get ready for Chrome 100 this time next year!) and the V8 team are stepping up to match it.

[Gatsby 3.0 Released](https://www.gatsbyjs.com/blog/gatsby-v3/) — Gatsby is a React-based open source framework with performance, scalability and security built-in. v3.0 promises an 80% speed improvement to the local development experience. They've also introduced [Gatsby Hosting](https://www.gatsbyjs.com/blog/introducing-gatsby-cloud-hosting/).

Hashim Warren (Gatsby)

[Electron 12.0.0 Released](https://www.electronjs.org/blog/electron-12-0) — Despite some compelling alternatives like [Neutralino](https://neutralino.js.org/) and [Tauri](https://tauri.studio/en/), Electron remains the big cheese in the *‘build desktop apps with JavaScript’* game. v12 updates from Node 12 to 14 (hello V8 8.9!) and to Chromium 89.

OpenJS Foundation and contributors

Quick Bits

- The [latest version of VS Code](https://code.visualstudio.com/updates/v1_54) has been released with stable Apple Silicon builds, TypeScript 4.2.2, and a 'quick fix' feature for [adding missing function declarations](https://code.visualstudio.com/updates/v1_54#_quick-fix-to-declare-missing-functions).
- Vue Mastery have opened up free access to their [Vue 3 + TypeScript course](https://www.vuemastery.com/free-week-ts/) till March 9 (next Tuesday).
- RedMonk is a top developer analyst firm and their [latest programming language rankings report](https://redmonk.com/sogrady/2021/03/01/language-rankings-1-21/) is fantastic news for JavaScript (which is the #1 language) and TypeScript (now at 8th).
- [Deno 1.8](https://deno.land/posts/v1.8) has landed with experimental support for WebGPU, making doing GPU-based computation from JavaScript even easier.
