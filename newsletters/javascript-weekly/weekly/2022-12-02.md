# JavaScript Weekly Issue 617: December 2, 2022

## Top Stories

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/kn5qxhrls2cd9olmbedl.jpg)

[Speeding Up the JS Ecosystem, One Library at a Time](https://marvinh.dev/blog/speeding-up-javascript-ecosystem/) — JavaScript projects are often so reliant upon dependencies that there are many opportunities to speed up swathes of the ecosystem by fixing small performance issues. Here, one of [Preact’s](https://preactjs.com/) developers shares how he’s been discovering such low hanging, yet hidden, fruit. Learning these techniques will take you (and us, hopefully) far.

Marvin Hagemeister

TC39 Updates: [The 93rd Meeting of Ecma TC39](https://github.com/tc39/agendas/blob/main/2022/11.md) — JavaScript's esteemed council of elders (not quite, but you get the idea) got together ([photographic proof](https://twitter.com/Jason_williams/status/1598072130100346880/photo/1)) over the past few days to advance various proposals. [Iterator Helpers](https://github.com/tc39/proposal-iterator-helpers), [Explicit Resource Management](https://github.com/tc39/proposal-explicit-resource-management) and [Set Methods](https://github.com/tc39/proposal-set-methods) all move to stage 3.

## 🛠 Code & Tools

[Neutralino.js 4.9: Lightweight Cross Platform Desktop App Framework](https://github.com/neutralinojs/neutralinojs/releases/tag/v4.9.0) — Think Electron but with a rather different approach: no embedded Chromium or Node.. it uses a system’s existing Web browser APIs, with the resulting pros and cons. v4.9 adds a new API supporting custom backend code in *any* language (that can communicate over a WebSocket). [Homepage](https://neutralino.js.org/).

Neutralinojs

[Mithril.js: Client-Side Framework for Single Page Apps](https://mithril.js.org/) — A neat alternative to things like Vue, React or Angular. It’s been around for years and deserves more attention we feel. Mithril is compact, fast, and runs closer to vanilla JS than the alternatives so is well suited for bringing together vanilla JS libraries. Want to compare it to `$your_favorite_framework`? [Here you go](https://mithril.js.org/framework-comparison.html).

Mithril

[A Professional Scheduling Component for React, Vue, Angular Apps](https://bryntum.com/products/schedulerpro/) — Schedule doctors, machines or your next trip to Mars. Easily done with the Scheduler Pro widget. Free 45-day trial.

Bryntum sponsor

[Vanilla Extract: Zero-Runtime Stylesheets in TypeScript](https://vanilla-extract.style/) — Using TypeScript as a preprocessor, you can use this framework agnostic approach to write type-safe, static CSS. Not my cup of tea, but the homepage does a good job on selling the idea.

SEEK

[Choices.js 10.2: A Configurable Select Box/Text Input Plugin](https://choices-js.github.io/Choices/) — Lots of examples or you can go straight to [the GitHub repo](https://github.com/Choices-js/Choices).

Josh Johnson

[Reapop 4.2: Customizable Notifications for React Apps](https://louisbarranqueiro.github.io/reapop/) — This project’s homepage is a big self demo. Keep clicking on “Random notification” to fill your view. [GitHub repo](https://github.com/LouisBarranqueiro/reapop).

Louis Barranqueiro
