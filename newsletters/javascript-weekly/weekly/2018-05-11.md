[Guess.js: A Toolkit for Using Data To Improve User Experiences](https://blog.mgechev.com/2018/05/09/introducing-guess-js-data-driven-user-experiences-web/) — An interesting project that can use data from Google Analytics combined with machine learning to intelligently pre-fetch resources. Addy Osmani [introduced it live](https://www.youtube.com/watch?v=Mv-l3-tJgGk#t=36m10s) at Google I/O earlier this week. There’s also a webpack plugin for quickly adding these abilities to React and Angular apps.

[ES Modules Enabled By Default in Firefox 60](https://bugzilla.mozilla.org/show_bug.cgi?id=1438139) — Firefox 60 went live this week and amongst [other changes](https://developer.mozilla.org/en-US/Firefox/Releases/60), [ES module support went live.](https://bugzilla.mozilla.org/show_bug.cgi?id=1438139) Lin Clark’s fantastic [deep dive](https://hacks.mozilla.org/2018/03/es-modules-a-cartoon-deep-dive/) is a great intro to what this means for JavaScript developers. This now means [all major, current browsers now support ES modules](https://caniuse.com/#feat=es6-module), hurrah!
