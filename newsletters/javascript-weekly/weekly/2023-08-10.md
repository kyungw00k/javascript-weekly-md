**⚡️ IN BRIEF:**

- *"Svelte 5 is going to be radical,"* [🐦 tweets](https://twitter.com/Rich_Harris/status/1688581184018583558) Rich Harris.
- Server side JS runtime [Deno 1.36 has been released](https://deno.com/blog/v1.36) with extended security options, a polyfill for `node:test`, completion of the `node:os` polyfill, and many other minor Node.js compatibility improvements.
- Google has unveiled a [sneak peek of a new IDE it's developing](https://developers.googleblog.com/2023/08/introducing-project-idx-experiment-to-improve-full-stack-multiplatform-app-development.html) called [Project IDX.](https://idx.dev/) It has VS Code at its base but with strong JS framework support, simulators, and lot of 'AI smarts' on top, [🐦 says Addy Osmani.](https://twitter.com/addyosmani/status/1688976969495683072)
- A revived but early stage [proposal for introducing observables](https://github.com/domfarolino/observable) to JavaScript. Your feedback is sought, so file an issue if you have any.
- A fun walk through front-end history with [a blog post containing every HTML element](https://www.patrickweaver.net/blog/a-blog-post-with-every-html-element/) *(well, almost – sorry* `applet`*).*
- [Nim 2.0 has been released.](https://nim-lang.org/blog/2023/08/01/nim-v20-released.html) Nim is a C++ and Python inspired, statically typed systems language, that [supports being compiled to JavaScript](https://nim-lang.org/features.html#javascript-compilation).

**🎉 RELEASES:**

- [TypeScript 5.2 RC](https://devblogs.microsoft.com/typescript/announcing-typescript-5-2-rc/) – TS 5.2 is nearly set for its final release. [Explicit resource management with `using`](https://github.com/tc39/proposal-explicit-resource-management) is the headline feature, but there's more, as usual.
- [esbuild 0.19.0](https://github.com/evanw/esbuild/blob/main/CHANGELOG.md#0190) – The high-perf bundler can now import paths containing wildcards and [supports advanced CSS `@import` rules.](https://github.com/evanw/esbuild/blob/main/CHANGELOG.md#01820) Note that this release contains *backwards-incompatible changes.*
- [Astro 2.10](https://astro.build/blog/astro-2100/) – There's also a [first beta of Astro 3.](https://astro.build/blog/astro-2100/#first-look-at-astro-30)
- [Node.js Security Releases](https://nodejs.org/en/blog/vulnerability/august-2023-security-releases) – v20.5.1 (Current), v18.17.1 (LTS) and v16.20.2 (LTS) are all part of this set of releases including OpenSSL security updates and fixes, mostly, around permission policies.
