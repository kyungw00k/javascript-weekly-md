The Undefined Podcast podcast

- The [first release candidate of Snowpack v3.0](https://www.snowpack.dev/posts/2020-12-03-snowpack-3-release-candidate) is out and there's a new way to load npm imports on-demand.. no more `npm install` step needed! We'll feature this again when it's released in January.
- Chrome 88 is in beta but [lots of changes](https://blog.chromium.org/2020/12/chrome-88-digital-goods-lighting.html) are forthcoming, including [Atomics](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Atomics), [SharedArrayBuffer](SharedArrayBuffer) (gated), the Digital Goods API, and the removal of Web Components v0 support.
- AWS Lambda (the serverless platform you can throw JavaScript - and more - at to have it run at your beck and call) now supports up to [10GB of memory and 6 vCPUS](https://aws.amazon.com/blogs/aws/new-for-aws-lambda-functions-with-up-to-10-gb-of-memory-and-6-vcpus/), [1ms billing granularity](https://aws.amazon.com/blogs/aws/new-for-aws-lambda-1ms-billing-granularity-adds-cost-savings/), and [functions in containers.](https://aws.amazon.com/blogs/aws/new-for-aws-lambda-container-image-support/)

- [A-Frame 1.1](https://github.com/aframevr/aframe/releases/tag/v1.1.0) — Web framework to build VR experiences. Now 5 years old.
- [Frappe Charts 1.5.4](https://github.com/frappe/charts) — Responsive, zero-dependency SVG charts.
- [webpack 5.9.0](https://github.com/webpack/webpack) — It splits. It shakes. It bundles.
- [Yarn 2.4](https://dev.to/arcanis/yarn-2-4-log-filters-audits-better-warnings-49io) — Package manager. Some updates on Yarn 3.0 here too.
- [Nashorn 15](https://mail.openjdk.java.net/pipermail/nashorn-dev/2020-November/007597.html) — Oracle's JDK-based JavaScript engine.
- [PWAdvent](https://pwadvent.dev/) is an advent calendar covering progressive browser API techniques, like the Web Share API, manifests, and service workers.
