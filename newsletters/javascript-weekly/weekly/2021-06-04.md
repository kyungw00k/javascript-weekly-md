Quick Bits

- [Chrome 92 is out in *beta*](https://blog.chromium.org/2021/06/chrome-92-web-apps-as-file-handlers-new.html) and the File Handling API now lets your PWAs define file handlers (origin trial only for now), V8 9.2 is included too with numerous JavaScript enhancements.
- [Chrome 92's DevTools](https://developer.chrome.com/blog/new-in-devtools-92/) are looking sharp too. As well as a CSS Grid editor, `const`s can now be redeclared if you're working in the console.
- A look at the stage 3 [`Array.prototype.at` proposal](https://github.com/tc39/proposal-relative-indexing-method) led to [some discussion on Hacker News](https://news.ycombinator.com/item?id=27367378).
- After only a brief moment in the sun, Node.js 15 has gone ['end of life.'](https://github.com/nodejs/Release#readme) Upgrade to Node 16 ASAP. Node has also just turned 12 years old 🎂
- The *Google Maps* JavaScript API has some [new powerful WebGL-powered functionality](https://cloud.google.com/blog/products/maps-platform/using-new-webgl-powered-maps-features) if you want to tilt, rotate, or work with 3D structures on your maps.
