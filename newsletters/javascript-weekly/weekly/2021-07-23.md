[JetBrains' 2021 Developer Ecosystem Survey Results](https://www.jetbrains.com/lp/devecosystem-2021/) — Over 30,000 developers took the [*State of Developer Ecosystem 2021*](https://www.jetbrains.com/lp/devecosystem-2021/) survey, with 70% being JavaScript users. You can find the [JavaScript-specific results here.](https://www.jetbrains.com/lp/devecosystem-2021/javascript/) Some interesting stats came of this:

- React is regularly used by 49% of respondents, with Vue.js at 43%.
- Go, TypeScript, and Python are the most common other languages JavaScript developers [plan to learn soon](https://www.jetbrains.com/lp/devecosystem-2021/#Do-you-plan-to-adopt--migrate-to-other-languages-in-the-next--months-If-so-to-which-ones).
- 42% claim to not to write unit tests regularly.
- Home working is, unsurprisingly, suddenly popular with 83% of developers WFH compared to 31% pre-pandemic.
