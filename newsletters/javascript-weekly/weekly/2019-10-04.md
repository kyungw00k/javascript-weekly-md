[Preact X: A Fast, 3KB React Alternative](https://github.com/preactjs/preact/releases/tag/10.0.0) — Two years in the making, the latest version of Preact, a popular lightweight React-compatible alternative, is here. [Node-RED 1.0 Released](https://nodered.org/blog/2019/09/30/version-1-0-released) — Node RED is a flow-based, visual programming tool (aimed primarily at hardware automation) that’s built on top of Node.js. Despite only reaching 1.0 now, it’s a mature project used in numerous real world IoT projects.

Nick O'Leary
