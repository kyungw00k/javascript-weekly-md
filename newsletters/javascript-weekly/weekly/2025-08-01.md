[Observable Notebooks 2.0 Technology Preview](https://observablehq.com/notebook-kit/) — The [Observable Framework](https://observablehq.com/framework/) and the new [Notebook Kit](https://observablehq.com/notebook-kit/kit) are just two parts of a rich ecosystem of [reactive](https://observablehq.com/@observablehq/learning-observable-reactive-dataflow) JavaScript ‘notebook’-style tools for creating data visualizations ([example](https://observablehq.com/@mbostock/hypnocontours)) and dashboards, originally created by [Mike Bostock](https://bost.ocks.org/mike/). This v2 release previews a big step forward with a new notebook file format based on HTML and, for the first time, support for true vanilla JavaScript, complete with the ability to import libraries with `import`. [Here’s another neat example](https://observablehq.com/notebook-kit/ex/d3/animated-treemap) showing off the potential. There are a lot of parts here, so dig in.

Observable, Inc.
