[Eliminating JS Framework Lock-in with Web Components?](https://jakelazaroff.com/words/web-components-eliminate-javascript-framework-lock-in/) — Can you build an app where each component is using a different framework? Amazingly, yes. Web components aren't new but are ['having a moment'](https://adactio.com/journal/20618) and Jake demos an interesting, if contrived, use case. He explains: *“Say you’re writing a Vue app and you really want to use a library that’s only available as a React component. You can wrap that library in a web component and use it in your Vue app...”*

Jake Lazaroff

[Announcing Deno Cron](https://deno.com/blog/cron) — Deno adds a cron-style feature to run code on a pre-defined schedule using `Deno.cron`. This works locally (behind `--unstable`) in a long-running process, but on *Deno Deploy* it analyzes the definition and runs the code on time, as you’d expect, with no extra work needed.

Zinkovsky and Jiang (Deno)
