# JavaScript Weekly Issue 547: July 16, 2021

## Top Stories

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60/bdmkhz0f54ua0ufrb5ti.jpg)

[V8 v9.2 and the `at` Function](https://v8.dev/blog/v8-release-92) — Every step forward the popular V8 JavaScript engine takes is news, but this is a relatively minor step with **the biggest practical addition being [the new '`at`' method](https://v8.dev/features/at-method) for Array, TypedArray and String.** Oh, and [Firefox 90 supports it](https://hacks.mozilla.org/2021/07/getting-lively-with-firefox-90/) too, plus a couple of other goodies.

[The Road to Ember 4.0](https://blog.emberjs.com/the-road-to-ember-4-0/) — The [Ember.js](https://emberjs.com/) framework is ten years old this year and while it’s not often in the headlines, it’s one of the most solid, reliable and battle tested systems out there and a big update is on the way.

[mem 9.0: Simple Memoization Library](https://github.com/sindresorhus/mem) — Memoization is an optimization technique used to speed up consecutive function calls by caching the result of calls with identical arguments.

Sindre Sorhus

[devcert: Development SSL/TLS Certificates Made Easy](https://www.npmjs.com/package/devcert) — Need a certificate for local development when using Node? This package takes an interesting approach of creating its own root authority and issuing certificates automatically against it. It’s so slick though, I was impressed by it in my tests.
