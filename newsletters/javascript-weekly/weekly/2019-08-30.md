[How JavaScript Grew Up and Became a 'Real' Language](https://t.co/ESSX9e8ATO) — If you were in the JavaScript space in the early 2000s, skip this one.. but I still remember the first time I saw Google Maps in 2005, then saw Microsoft’s documentation for `XMLHttpRequest` (which shipped in IE 5 in 1999!), and a Pandora’s box was rapidly opened as developers finally saw potential in making Web pages interactive in a truly two-way fashion. JavaScript’s *second wind* had begun!

[Should You Be Using Web Workers? (Probably Not.)](https://t.co/tjRrj50Tgt) — An interesting contrast to Surma’s [recently popular article promoting Web Workers](https://dassur.ma/things/when-workers/) and their ability to execute JavaScript off of the main thread.
