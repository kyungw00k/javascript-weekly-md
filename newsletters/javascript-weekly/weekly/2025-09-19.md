# JavaScript Weekly Issue 753: September 19, 2025

## Top Stories

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/kpbxelfljzef17wijism.jpg)

[The 'Shai-Hulud' npm Supply Chain Attack Rumbles On](https://socket.dev/blog/ongoing-supply-chain-attack-targets-crowdstrike-npm-packages) — Now named after a term for the [sandworms](https://en.wikipedia.org/wiki/Sandworm_(Dune)) of the Dune universe, the ongoing malicious supply chain attack affecting the npm ecosystem has grown in scale with hundreds of packages affected in an attempt to exfiltrate tokens and secrets from developers’ machines.

Pandya, van der Zee, and Brown (Socket)

The story above has triggered a wave of responses and mitigations:

- [pnpm 10.16 has been released](https://pnpm.io/blog/releases/10.16) with support for delaying dependency updates using a `minimumReleaseAge` option – [explained in more depth here](https://pnpm.io/supply-chain-security). We expect to see more tools adopt similar features in due course.
- In ['a meditation on npm supply chain attacks'](https://tane.dev/2025/09/oh-no-not-again...-a-meditation-on-npm-supply-chain-attacks/), Tane Piper points a finger at Microsoft's role.
- Drew DeVault talks about [a better future for JavaScript 'that won't happen'](https://drewdevault.com/2025/09/17/2025-09-17-An-impossible-future-for-JS.html), focusing on the systemic changes needed to improve the situation.

⚖️ [Deno Asks: 'Help Us Raise $200k to Free JavaScript from Oracle'](https://deno.com/blog/javascript-tm-gofundme) — The term JavaScript(™) is actually an Oracle trademark, but Ryan Dahl and Deno are [attempting to fight that](https://javascript.tm/) and are raising funds to get through the all-important discovery phase of the cancellation petition.

Ryan Dahl (Deno)

**IN BRIEF:**

- [The WebAssembly 3.0 standard is now 'live'](https://webassembly.org/news/2025-09-17-wasm-3.0/) (and already ships with most browsers) with various features that should improve JS interoperability, including garbage collection, tail calls, exception handling, and ways to manipulate JavaScript strings directly.
- ▶️ Vue's [Evan You was recently interviewed](https://www.youtube.com/watch?v=FS0Ds0nIC8E) by Nuno Maduro of the Laravel project about his background at Google, how Vue.js was first created, the relationship between Next.js and Nuxt, and his current work with [void(0).](https://voidzero.dev/)
- We recently mentioned [Andromeda](https://tryandromeda.dev/) as being the newest JavaScript runtime to appear, but [now there's Ion](https://github.com/alshdavid/ion), a runtime aimed at bringing a JavaScript engine into Rust programs.
- [Webflow has donated $150k to the Astro project.](https://astro.build/blog/webflow-official-partner/)

**RELEASES:**

- [Safari 26.0 has been released](https://webkit.org/blog/17333/webkit-features-in-safari-26-0/) alongside macOS 26.0, iOS 26.0, etc. Along with numerous CSS enhancements and a new `<model>` element for embedding 3D models onto web pages, every site can now "be a web app" on iOS and iPadOS if a user adds it to their home screen.
- [Bun v1.2.22](https://bun.com/blog/bun-v1.2.22) – Stack traces now include asynchronous call frames, there's `Bun.YAML.stringify` to turn objects into YAML, bundler & minifier improvements, and more.
- As of React Router 7.9.0, the [long-awaited middleware feature is now stable.](https://remix.run/blog/middleware)
