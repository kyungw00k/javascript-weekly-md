## 🔧 Code & Tools

[Stryker: Test Your *Tests* with Mutation Testing](https://stryker-mutator.io/) — Stryker fiddles with your tests (in a large and complicated number of ways) and expects this to *break* them. If it doesn’t, your tests are too brittle and therefore failed the test. This is not something to run frequently (as it can be very slow) but is an interesting way to stress test your tests, if you will.

Jansen, de Lang, et al.

[Immer 6.0: The Popular Immutable State Library](https://github.com/immerjs/immer) — Immer only continues to get better over time but take note of the [breaking changes.](https://github.com/immerjs/immer/releases/tag/v6.0.0)

immer
