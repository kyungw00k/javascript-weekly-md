## Top Stories

📖 [Exploring JavaScript (ES2025 Edition)](https://exploringjs.com/js/) — Dr. Axel is back with his latest book covering all things relating to modern JavaScript at the language level (think built-in data types, modularity, how objects, classes and promises work, etc.). As with all of Axel's books, it’s available to buy but also to [read online in HTML form for free](https://exploringjs.com/js/book/index.html). He’s also produced [a set of flashcards](https://exploringjs.com/js/downloads/exploring-js-cards-topics-preview.html) to help you learn language features in both HTML and Anki forms.
