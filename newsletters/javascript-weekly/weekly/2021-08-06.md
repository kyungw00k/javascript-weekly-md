**QUICK BITS:**

- There's [a new Visual Studio Code update](https://code.visualstudio.com/updates/v1_59) – the Live Preview extension now supports [JavaScript debugging within previews.](https://code.visualstudio.com/updates/v1_59#_live-preview)
