[Ryan Dahl Asks Oracle to Release the 'JavaScript' Trademark](https://tinyclouds.org/trademark) — It's in our small print, but did you know Oracle owns the 'JavaScript' trademark? This is why the standard is called *ECMAScript* instead. It’s all down to Netscape’s deal with Sun Microsystems, which Oracle later acquired, and is infamously poorly defended (Brendan Eich suggested many years ago that he’s not sure it *could* be defended.) So if you’ve got Larry Ellison’s phone number, see if he’ll do us all a solid 🙏

[Introducing Signals: A Reactive State Primitive That's Fast By Default](https://preactjs.com/blog/introducing-signals/) — Signals provide a reactive way of expressing state (in a very natural way that feels like using plain values) such that apps stay fast regardless of complexity. It’s aimed at Preact, the [slimmer alternative to React](https://preactjs.com/), but is possible to use with regular React by way of monkeypatching.

[El: A Tiny Web Component-Based UI Framework](https://github.com/frameable/el) — The very definition of short and sweet. You get a lot in 150 lines, including a built-in observable store and reactive templates with one-way binding. Think a very lightweight React/Vue mixed with Lit. Here’s a [one file example](https://github.com/frameable/el/blob/main/example.html) of creating a todo-list app on top of it.
