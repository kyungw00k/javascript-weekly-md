# JavaScript Weekly Issue 518: December 11, 2020

## Top Stories

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60/ym0qviypnlrw97o4vlyw.jpg)

[The 'Import on Interaction' Pattern](https://addyosmani.com/blog/import-on-interaction/) — You know what’s faster than loading lots of front-end code? *Not* loading it. You can then ‘lazy-load’ non-critical resources when the user needs to actually use them. Addy Osmani digs in deep as to how this can work here.

Addy Osmani

[The JavaScript Christmas Advent Calendar 2020](https://javascript.christmas/2020) — *Bekk* has run these advent calendars on a variety of topics for a few years now and they’re a good source of bite size articles, tips, etc. You might enjoy [The Three Ways of Fetching](https://javascript.christmas/2020/2), [What’s New in ES2021](https://javascript.christmas/2020/7) or [Machine Learning with JavaScript](https://javascript.christmas/2020/5) so far this year.

Bekk

[The Easiest Way to Monitor Node: Automatic Instrumentation](https://blog.appsignal.com/2020/12/08/the-easiest-way-to-monitor-nodejs-automatic-instrumentation.html?utm_source=javascript-weekly-sponsored&utm_medium=email&utm_campaign=2020-12-08) — Monitoring your Node.js apps can be hard. [Get Ready to Up Your (Google) Apps Script](https://developers.googleblog.com/2020/12/get-ready-to-up-your-apps-script.html) — Google introduced *Apps Script* over ten years ago as a powerful, JavaScript-powered way to script its various services (e.g. Gmail, Sheets..) Rather than shutting it down, Google has unveiled a new online IDE to make the development process far smoother.

Google

[Deno 1.6 Released: You Can Now Build Executables](https://deno.land/posts/v1.6) — Perhaps the biggest post-1.0 release for the security-oriented TypeScript-leaning JavaScript runtime as it can now *build self contained, standalone binaries* though there are limitations and drawbacks (one being the file sizes). A promising step forward for JS tooling, though? Certainly.

Iwańczuk, Casonato, and Dahl

⚡️ Quick Releases

- [Stimulus 2.0](https://discourse.stimulusjs.org/t/announcing-stimulus-2-0/1482) — 'Decorate your HTML' with JS functionality library from Basecamp — commonly associated with Ruby on Rails.
- [Fable 3.0](https://fable.io/blog/Announcing-Nagareyama-4.html) — A F# to JavaScript compiler.
- [μPlot 1.5](https://github.com/leeoniya/uPlot) — High performance time series plotting.
- [mapbox-gl-js 2.0](https://github.com/mapbox/mapbox-gl-js/releases/tag/v2.0.0) — Tiled WebGL-powered maps. Note the license change.
- [Pickr 1.8.0](https://github.com/Simonwep/pickr) — No depedency color picker library.
- [jsPDF 2.2](https://github.com/MrRio/jsPDF) — Client-side PDF generation.
- The [State of JavaScript 2020 Survey](https://stateofjs.com/?source=js_weekly) is back again for another year. The results of this survey are always fascinating and there's been a lot of buzz on social media about the questions this year.
- Sebastian McKenzie, the creator of the [Rome](https://rome.tools/) toolkit, [needs your help in funding the project](https://rome.tools/funding/) now he's working on it full time. He also shares a lot of updates on what he's working on.
- The [November 2020 update of VS Code](https://code.visualstudio.com/updates/v1_52) is out — you can now set default locations of Node.js and Chrome binaries if you want to debug with different versions.
- If you didn't hear, GitHub has an optional ['dark mode' theme](https://github.com/settings/appearance) now.
- Someone who was only 1 year old when Brendan Eich first developed JavaScript has [gone back and compiled an early JavaScript engine](https://github.com/doodlewind/mocha1995/blob/main/blog/about.md) and [put it online](https://mocha1995.js.org/) for you to play with. *Trés cool.*
- NativeScript has [joined the OpenJS Foundation.](https://www.infoq.com/news/2020/12/nativescript-joins-openjs/)
- A fantastic gist and comment thread of people [code-golfing a Flappy Bird implementation](https://gist.github.com/gullyn/95b2ab9e465317f1d4e4607cf6e94205) down to under 200 bytes. [Playable here.](https://jsfiddle.net/z3bmtg5u/1/)
