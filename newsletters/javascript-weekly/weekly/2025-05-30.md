## Top Stories

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/qouongcfhjdkbdpc7gyz.jpg)

[*Wake Up, Remix!* Everything's Changing..](https://remix.run/blog/wake-up-remix) — Big news from the [Remix](https://remix.run/) camp this week. About a year ago, [Remix and React Router merged together](https://remix.run/blog/merging-remix-and-react-router) reflecting their shared goals and code, but now it’s *all change* again. [React Router](https://reactrouter.com/) is now basically what Remix originally intended to be, and so ‘Remix’ is rebooting as a model-first, low-dependency, Web API-centric full-stack framework built on [Preact](https://preactjs.com/). It’ll no longer be a 'React framework' *per se*.

Michael Jackson and Ryan Florence

🕒 [The Upcoming Temporal API and What Problems It Will Solve](https://waspdev.com/articles/2025-05-24/temporal-api) — The [Temporal API](https://tc39.es/proposal-temporal/docs/) has been cooking for many years now as a new way to work with dates and times in JavaScript. It’s just been [enabled in Firefox 139 by default](https://spidermonkey.dev/blog/2025/04/11/shipping-temporal.html) and sure to come to more runtimes ASAP. Here’s why it matters and what it offers.

Suren Enfiajyan

**IN BRIEF:**

- Matteo Collina has unveiled [php-node](https://github.com/platformatic/php-node), a way to run PHP apps within the same process as Node.js. Run WordPress with a Next.js frontend? Sure.
- [TC39's 108th meeting](https://github.com/tc39/agendas/blob/main/2025/05.md) is currently taking place with many proposals being advanced, such as [Seeded Pseudo-Random Numbers](https://github.com/tc39/proposal-seeded-random) moving to stage 2 and [`Error.isError`](https://github.com/tc39/proposal-is-error) moving to stage 4.
- [The OpenJS Foundation is now a CVE Numbering Authority (CNA)](https://socket.dev/blog/openjs-foundation-is-now-a-cna) for over 40 popular JavaScript projects it hosts, including ESLint, Express, and Electron.
