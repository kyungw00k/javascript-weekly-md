[JavaScript's Creator Recommends We Call JavaScript 'JS'](https://mobile.twitter.com/BrendanEich/status/1278803332287483904) — In response to a lively Twitter thread this week posing the question [*“If you were given the chance to rename JavaScript, what would you call it?”*](https://mobile.twitter.com/runhappylife/status/1278360272231641088) Brendan Eich [wisely suggested](https://mobile.twitter.com/BrendanEich/status/1278794515973693440), nay *declared*, we go with *JS*. While there’s an aspect of fun around this whole debate, it seems a sensible idea given JS has no relation to Java at all, although it might take us a while to rebrand ;-)
