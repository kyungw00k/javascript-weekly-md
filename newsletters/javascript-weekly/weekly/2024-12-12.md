# JavaScript Weekly Issue 716: December 12, 2024

## Top Stories

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/pwf3ggjktro0rsmoorh7.jpg)

[Boa v0.20: An Alternative JavaScript Compiler](https://boajs.dev/blog/2024/12/05/boa-release-020) — Under development for several years, Boa has a few missions: be a Rust ECMAScript implementation, be easy to embed in Rust projects, and be a fast, safe JS engine overall. v0.20 sees a bump up to 89.92% compliance in the [Test262 suite](https://test262.fyi/#), improves Temporal support, adds `Atomics.pause`, and more. This is no toy engine.

Boa Developers

[React v19 Now Stable](https://react.dev/blog/2024/12/05/react-19) — First teased in [a February 2024 update](https://react.dev/blog/2024/02/15/react-labs-what-we-have-been-working-on-february-2024), React 19 has been in beta for 8 months, so there’s already a wealth of content about it, from the [React 19 upgrade guide](https://react.dev/blog/2024/04/25/react-19-upgrade-guide) or an [explanation of its compiler](https://react.dev/learn/react-compiler) to [cheat sheets](https://www.epicreact.dev/react-19-cheatsheet) and Vercel’s roundup of [what’s new in React 19.](https://vercel.com/blog/whats-new-in-react-19)

🖼️ [wasm-vips: `libvips` Image Processing via WebAssembly](https://github.com/kleisauke/wasm-vips) — [libvips](https://www.libvips.org/) is an efficient image processing library written in C. [Civet 0.9: Code More with Less Using a TypeScript Superset](https://civet.dev/) — Two years old and well maintained, Civet offers an interesting approach.
