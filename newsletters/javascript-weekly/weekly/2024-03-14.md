## Top Stories

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/moq1oa1istublivmxoai.jpg)

[Use Parcel Macros in Other Bundlers with `unplugin-parcel-macros`](https://github.com/devongovett/unplugin-parcel-macros) — [Macros](https://parceljs.org/features/macros/), as implemented in Parcel ([and Bun!](https://bun.sh/blog/bun-macros)), are JS functions that run at *build* time whose results are *inlined* into a bundle in place of the original call. You can now use this feature with webpack, Vite, Rollup, esbuild, etc. [Here's a quick example](https://github.com/devongovett/unplugin-parcel-macros?tab=readme-ov-file#example) of how handy this can be.

Devon Govett

𝕏 Devon goes into extra detail [🐦 in this Twitter/X thread.](https://twitter.com/devongovett/status/1767939537907462211) You can also revisit [the release of Parcel 2.12.0](https://parceljs.org/blog/v2-12-0/) to learn more.

[Shiki 1.0: A Powerful Syntax Highlighter](https://shiki.style/) — A few months ago, we linked to [Shikiji](https://shikiji.netlify.app/), a fork of Shiki that was created to push the project forward.
