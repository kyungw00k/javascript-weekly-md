[Metaprogramming with the 2022-03 Decorators API](https://2ality.com/2022/10/javascript-decorators.html) — If you’re a Python developer, you’ll be miles ahead on this idea. If not, *decorators* provide a way to manipulate classes, fields, methods, and accessors *at define time* to give them extra, amended, or wrapped functionality or behavior. [The proposal](https://github.com/tc39/proposal-decorators) is at stage 3 at TC39, is supported by Babel, and will soon be supported by TypeScript, so why not get up to speed with Dr. R.

Dr. Axel Rauschmayer

[Node.js 19 (Current) Released](https://nodejs.org/en/blog/announcements/v19-release-announce/) — We did a big feature in [yesterday’s *Node Weekly*](https://nodeweekly.com/issues/459), but in short, this is the new, edgiest 'Current' non-LTS Node.js release which gains a Nodemon-esque `--watch` mode, HTTP KeepAlive on by default, stable WebCrypto API, and a bump to V8 10.7. Node 18 goes Active LTS next week.

[A Real World React to htmx Migration Story](https://htmx.org/essays/a-real-world-react-to-htmx-port/) — [htmx](https://htmx.org/) is a library that ‘turbo charges’ HTML by letting you use numerous JS and Web API features via custom HTML attributes. [p5.js 1.5: Creative JS Coding, Now with Animated GIF Exporting](https://p5js.org/) — p5 is hard to define – it’s like [Processing](https://processing.org/) but for JavaScript, and it lets you create creative, visual experiences in the browser. Examples? [Why, of course.](https://p5js.org/examples/) v1.5 is a significant release as you can now easily [save GIFs of your creations](https://p5js.org/reference/#/p5/saveGif) which leads to [fantastic tweets like this.](https://twitter.com/andrewmcwhae/status/1583258220692672512) *Basically if you want to generate cool animated GIFs using JavaScript, this should be your first port of call.*
