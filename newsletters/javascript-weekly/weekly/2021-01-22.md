# JavaScript Weekly Issue 522: January 22, 2021

## Top Stories

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60/gegtb5t4g2cxvkc20g25.jpg)

[The JavaScript 'Rising Stars' of 2020](https://risingstars.js.org/2020/en) — Fingers crossed this is our final ‘look back’ at 2020, but the ‘Best of JS Rising Stars’ roundup comes out at the start of each year and shows off the fastest growing JavaScript tools and projects over the past year based on GitHub stars (which, yes, is *by no means a definitive metric*). Stand out projects include [Deno](https://deno.land/), [Playwright](https://github.com/microsoft/playwright), [esbuild](https://github.com/evanw/esbuild), and [Rome](https://rome.tools/), all of which we expect to see great things of in 2021.

Michael Rambeau and Benjamin Blackwood

[EStimator.dev: The Modern JavaScript Savings Calculator](https://estimator.dev/) — Enter a URL and this tool will tell you how much faster the site could be if the code was switched over (or compiled) to modern JavaScript. Read about it [here](https://web.dev/publish-modern-javascript/).

Google Chrome Labs

![](https://copm.s3.amazonaws.com/ce33b14c.png)

[Spreadsheet Viewer: Render XLSX Files In the Browser](https://handsontable.com/spreadsheet-viewer/) — Spreadsheet Viewer is a client-side component that allows users to preview XLSX file without the need of opening it in an external app. It is mobile-friendly, and supports all major frameworks such as React, Angular, and Vue. Check out the [live demo](https://handsontable.com/spreadsheet-viewer/).

Handsontable sponsor

[JerryScript: A Ultra-Lightweight JS Engine for the Internet of Things](https://github.com/jerryscript-project/jerryscript) — If you’ve got a constrained environment (e.g. a microcontroller, a watch, or the like) with only kilobytes of RAM to hand, this is the sort of thing you might want to try. It’s ES 5.1 compliant and, as of this week, supports [realms](https://github.com/tc39/proposal-realms), BigInt, class fields, `flatMap`, [and more](https://github.com/jerryscript-project/jerryscript/releases/tag/v2.4.0).

JerryScript

[How I Build JavaScript Apps in 2021](https://timdaub.github.io/2021/01/16/web-principles/) — A freelance developer shares his development approach from *avoiding* build processes and transpiling, to testing and re-using his own work.

Tim Daubenschütz
