**IN BRIEF:**

- Angular's [ngConf 2024](https://ng-conf.org/) is taking place *right now* and some interesting announcements have been coming out, including [some AI integrations](https://twitter.com/AngularUniv/status/1770489729621909679) and [a merger with another Google framework](https://twitter.com/sarah_edo/status/1770478763253379488) called *Wiz.*

## 📒 Articles & Tutorials

📉 [Optimizing JavaScript for Fun and for Profit](https://romgrk.com/posts/optimizing-javascript) — A tour of several things to avoid to keep your code running fast and smooth, complete with examples. While the performance ground upon which we walk is often shifting, many of these things are good, common sense practices.

Rom Grk

[Bringing `require(esm)` to Node.js](https://joyeecheung.github.io/blog/2024/03/18/require-esm-in-node-js/) — Joyee writes about landing [experimental support for `require()`-ing synchronous ES modules](https://github.com/nodejs/node/pull/51977) in Node, a feature that has been a long time coming.

Joyee Cheung

[FusionAuth’s New Vue, React And Angular SDKs](https://fusionauth.io/blog/javascript-sdks?utm_medium=newsletter&utm_source=jsweekly&utm_campaign=mar24_newsletter) — These JavaScript SDKs simplify authentication integration into React, Angular, or Vue apps, allowing you to focus on user features while ensuring top-tier security.

FusionAuth sponsor

[How Figma Built a Custom Permissions DSL](https://www.figma.com/blog/how-we-rolled-out-our-own-permissions-dsl-at-figma/) — Figma has a complex permissions setup, the implementation of which was causing tech debt, bugs, and delays. After not finding any open source answers, they built a DSL that isolated policies and data while also being cross-platform (Ruby and TypeScript.)

Jorge Silva (Figma)

[Building a Node App with TypeScript](https://www.totaltypescript.com/typescript-and-node) — How to set up a seamless TypeScript development experience for bundling modern Node apps with pnpm and ES modules, by someone who certainly knows a few things about TypeScript.

[Type Predicate Inference: The TS 5.5 Feature No One Expected](https://www.totaltypescript.com/type-predicate-inference) — And another from Matt, who’s getting excited about the forthcoming inclusion of type predicate inference from function bodies in TypeScript 5.5.

## 🛠 Code & Tools

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/aeajv2segczslpjgwlwv.jpg)

[Atrament 4.0: Library for Smooth Canvas Drawing and Handwriting](https://www.fiala.space/atrament/demo/) — A small library for beautiful drawing and handwriting on canvas elements. It’s been around for several years but [v4.0](https://github.com/jakubfiala/atrament/releases/tag/v4.0.0) is almost a complete rewrite. [GitHub repo](https://github.com/jakubfiala/atrament).

Jakub Fiala

[MistCSS: Write React Components with Just CSS](https://typicode.github.io/mistcss/) — Forget about CSS-in-JS.. what about JS-*from*-CSS? This approach provides a way to compile CSS-only React components into a TypeScript equivalent. ([Code example.](https://typicode.github.io/mistcss/getting-started.html))

Typicode

[MicroDiff: No Dependency Object and Array Comparison Library](https://github.com/AsyncBanana/microdiff) — Given two objects or arrays, it returns the differences (think `diff` but for JavaScript objects). Boasts high performance and TypeScript support. There’s also an article from 2022 [about how it works](https://byteofdev.com/posts/microdiff/).

AsyncBanana

[Transformational Auth & Identity | Userfront](https://userfront.com/?&utm_source=javascriptweekly.com&utm_medium=newsletter&utm_campaign=sponsored_link&utm_content=transformational) — Userfront is an auth & identity platform that transforms your business to serve free, freemium and enterprise customers.

Userfront sponsor

[oneRepo 1.0: A New Suite of Monorepo Tools for Teams](https://onerepo.tools/) — There always seems to be discontent when JS monorepo management tools get discussed, but maybe [Paul’s approach could win you over?](https://paularmstrong.dev/blog/2024/02/27/2024-02-27-introducing-onerepo-the-javascript-typescript-monorepo-toolchain-for-safe-strict-fast-development/)

Paul Armstrong

[Eta 3.4: Embedded JS Template Engine for Node, Deno, & Browsers](https://eta.js.org/) — Boasts being lighter and faster than EJS but with many of the same features (it looks a *lot* like Ruby’s ERB, if you like that vibe). [GitHub repo](https://github.com/eta-dev/eta).

Ben Gubler

[TanStack Virtual 3.2: Headless UI for Virtualizing Large Element Lists](https://tanstack.com/virtual/latest) — Supporting TS/JS, React, Solid, Svelte and Vue, this is a way to build 60 fps experiences even with large numbers of elements while retaining full control over markup and style.

Tanner Linsley

[React Data Grid: A Fast Excel-Style Data Grid Component](https://adazzle.github.io/react-data-grid/#/common-features) — A fantastic live demo shows off all of this component’s unique features, from grouping rows and columns to [handling a million cells](https://adazzle.github.io/react-data-grid/#/million-cells) without slowdowns. Works with React 18+. [GitHub repo](https://github.com/adazzle/react-data-grid).

Adazzle
