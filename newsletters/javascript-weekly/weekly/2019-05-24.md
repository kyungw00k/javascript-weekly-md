[Functional JavaScript: How to Use `reduce` for More Than Just Numbers](https://jrsinclair.com/articles/2019/functional-js-do-more-with-reduce/) — Eric Elliott calls it *“one of the most versatile functions that was ever discovered”* but a lot of developers struggle with `reduce` beyond basic use cases. If that includes you, this tutorial goes into more depth.

James Sinclair

[Pixi.js 5.0: Create Beautiful 2D Web Experiences](https://github.com/pixijs/pixi.js) — Boasts the ‘fastest, most flexible 2D WebGL renderer’ to let you take advantage of hardware acceleration without getting involved in WebGL or 3D concerns. Check out [demos](https://pixijs.io/examples/#/demos-basic/container.js) for what the code looks like and what you’d use it for. There’s also a [Pixi Playground](https://www.pixiplayground.com/#/edit) for quickly crafting your own experiments.

PixiJS

[Tornis: Watch and Respond to Changes in the Browser's Viewport](https://tornis.robbowen.digital/) — A JavaScript library that tracks pointer position, viewport size, scroll position, etc. and lets you perform actions based upon them.

Robb Owen

[Node 12.3 Released with WebAssembly Module Support](https://nodejs.org/en/blog/release/v12.3.0/) — It’s experimental, but use `--experimental-wasm-modules` to be able to do, say, `import './module.wasm'`. Here’s [some documentation on how they work](https://github.com/nodejs/node/blob/bbc254db5db672643aad89a436a4938412a5704e/doc/api/esm.md) if you want to give it a try at this early stage.

Node.js Foundation

[WebAssembly at eBay: A Real-World Use Case](https://www.ebayinc.com/stories/blogs/tech/webassembly-at-ebay-a-real-world-use-case/) — Notes on how the eBay engineering team approached building a better barcode scanner using WebAssembly to replace a pure JavaScript version.

eBay Engineering
