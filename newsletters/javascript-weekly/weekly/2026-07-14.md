[Rewriting Bun in Rust](https://bun.com/blog/bun-in-rust) — The creator of Bun shares the full tale of Bun being ported from Zig to Rust, with the assistance of many Claude Code instances churning through ~$165k in usage (at API pricing). The Rust version forms the basis of Bun 1.4, expected to be released any day now.

Jarred Sumner

[npm v12 Released: Install Scripts are Off by Default](https://github.blog/changelog/2026-07-08-npm-install-time-security-and-gat-bypass2fa-deprecation/) — npm 12 is now GA with [many breaking changes](https://github.com/npm/cli/releases/tag/v12.0.0). The big change is lifecycle scripts no longer run by default, but here are [more practical implications to consider](https://socket.dev/blog/npm-12). It may also be worth upgrading [via 11.18.0](https://github.com/npm/cli/releases/tag/v11.18.0) first.

GitHub

**IN BRIEF:**

- TC39's 115th meeting takes place next week. [Here's the agenda](https://github.com/tc39/agendas/blob/main/2026/07.md), including [Await Dictionary](https://github.com/tc39/proposal-await-dictionary) aiming for Stage 3, plus [Decimal](https://github.com/tc39/proposal-decimal/), Import Defer and [Map.take](https://github.com/dcrousso/proposal-Map-take).
- [Next.js now has a formal security release program](https://nextjs.org/blog/next-security-release-program) with monthly announcements and scheduled releases. The first is expected on July 20.
- ⚠️ The `jscrambler` npm package [suffered a supply chain attack at the weekend](https://socket.dev/blog/jscrambler-supply-chain-attack) which Socket detected within six minutes.
- The [Porffor](https://porffor.dev/) ahead-of-time JavaScript compiler [is now self-hosting](https://bsky.app/profile/honk.foo/post/3mq5u64sl4k27) (can compile itself with itself).
- [Vercel has acquired](https://vercel.com/blog/vercel-acquires-better-auth) the team behind [Better Auth](https://better-auth.com/), the popular auth library.

**RELEASES:**

- ⭐ [TypeScript 7.0](https://devblogs.microsoft.com/typescript/announcing-typescript-7-0/) – The final release of the "10x faster" Go-powered version of the TypeScript compiler is here. A (temporary) lack of a full API means many users are recommended to [stay on 6.0](https://devblogs.microsoft.com/typescript/announcing-typescript-7-0/#typescript-and-embedded-languages) for now, though.
- [Node.js 26.5.0 (Current)](https://nodejs.org/en/blog/release/v26.5.0) – Adds support for [importing text files in ES modules](https://github.com/tc39/proposal-import-text) via import attributes, plus `blob.textStream()`.
- [ESLint 10.7](https://eslint.org/blog/2026/07/eslint-v10.7.0-released/), [Storybook 10.5](https://github.com/storybookjs/storybook/releases/tag/v10.5.0), [Vite DevTools 0.4](https://github.com/vitejs/devtools/releases/tag/v0.4.0)
