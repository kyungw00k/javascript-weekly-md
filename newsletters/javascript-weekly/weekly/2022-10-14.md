[`Intl` Explorer: A Way to Learn and Experiment with the ECMAScript Internationalization API](https://www.intl-explorer.com/) — Supported by all major browsers, the [`Intl` object](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl) provides access to the *ECMAScript Internationalization API*, a suite of functions for language sensitive string comparison, number formatting, and more. This site provides an interactive way to learn about how it operates.

Jesper Orb

▶ [Creating Your Own JavaScript Runtime with V8](https://www.youtube.com/watch?v=ynNDmp7hBdo) — Want to join the pantheon of developers famed and esteemed for creating things like Node, Deno, and Bun by creating a JS runtime of your own? Erick takes us deep into the main moving parts and shows us how to tie them together for ourselves. While you might not build the next Deno or Bun for yourself, there’s a *lot* to learn about JS runtimes here.

Erick Wendel

[Node v18.11.0 (Current) Released; Gains 'Watch Mode'](https://nodejs.org/en/blog/release/v18.11.0/) — Node’s latest release doesn’t gain a lot, but does gain an experimental `--watch` feature that automatically restarts the running process when imported files are changed (reminiscent of [nodemon](https://github.com/remy/nodemon)) – this feature [was discussed at length](https://github.com/nodejs/node/issues/40429) recently.

[Lerna Reborn: What’s New in Version 6?](https://blog.nrwl.io/lerna-reborn-whats-new-in-v6-10aec6e9091c) — Far from being obsolete or deprecated, the Lerna monorepo-oriented JavaScript build system is taking big strides forward under Nrwl’s stewardship. v6 gains a lot of speed with efficient task scheduling and caching by default, a VS Code extension, Prettier support, and more.
