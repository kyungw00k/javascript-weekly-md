**IN BRIEF:**

- The [latest VS Code update](https://code.visualstudio.com/updates/v1_68) is out. TypeScript 4.7 is included and, more excitingly, **Go to Definition** [now jumps](https://code.visualstudio.com/updates/v1_68#_go-to-source-definition) to the JS implementation of functions and symbols in external libraries rather than the type definitions.
- 😆 Last week we linked to a guy who made a graphical game in Word with JavaScript using *Office Add-Ins* - he's back again [▶️ implementing Wordle in Word.](https://www.youtube.com/watch?v=N2btcfPtxQ0) It's nice to see someone having some fun.
- **DOOM?** We recently linked to [Sliderland](https://sliderland.blinry.org/), a curious minimalist JS coding playground – its creator has now created [a DOOM-like game](https://blinry.org/sliderland/doom/) on top of it(!)
- **SAFARI:** Apple's developer conference, WWDC, is currently underway and [new WebKit/Safari features](https://webkit.org/blog/12824/news-from-wwdc-webkit-features-in-safari-16-beta/) are on show in Safari 16 Beta. There's support for *Web Inspector Extensions* and support for [*Shared Workers*](https://developer.mozilla.org/en-US/docs/Web/API/SharedWorker), service workers that can be shared between browsing contexts. We can expect [Web Push](https://webkit.org/blog/12945/meet-web-push/) support in 2023, too.
- [Astro 1.0 was expected this week](https://astro.build/blog/astro-1-release-update/) but the release of [the novel static site builder](https://astro.build/blog/introducing-astro/) is being pushed back to late July.
- GitHub is introducing [the idea of Achievements](https://github.blog/2022-06-09-introducing-achievements-recognizing-the-many-stages-of-a-developers-coding-journey/) where you earn badges for achieving as yet undefined goals.
