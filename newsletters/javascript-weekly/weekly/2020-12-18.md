⚡️ Quick Releases

- [npm 7.2.0](https://blog.npmjs.org/post/637599402762141696/release-v720) — Nomenclature predictably misunderstood.
- [Node 14.15.3](https://nodejs.org/en/blog/release/v14.15.3/) — The popular server-side runtime.
- [Popper.js 2.6](https://popper.js.org/) — Tooltip and popover placement engine.
- [esprint 2.0](https://github.com/pinterest/esprint) — Fast multithreaded ESLint runner.
- [Polly.js 5.1.0](https://netflix.github.io/pollyjs/) — Record, replay, and stub HTTP interactions.
- [Flux 4.0](https://github.com/facebook/flux/releases/tag/4.0.0) — React app architecture library but now in 'maintenance mode' as of this release.
- Node.js has a full suite of [security releases coming](https://nodejs.org/en/blog/vulnerability/january-2021-security-releases/) in the week of January 4, 2021 due to some newly discovered security issues.
- The npm registry now displays [when packages have bundled TypeScript declarations.](https://github.blog/changelog/2020-12-16-npm-displays-packages-with-bundled-typescript-declarations/) See [Fastify's package page](https://www.npmjs.com/package/fastify) for an example.
- GitHub has [got rid of its cookie banners](https://github.blog/2020-12-17-no-cookie-for-you/) — how? By ceasing to use all non-essential cookies. Fantastic!
- The [State of JS Survey](https://stateofjs.com/?source=js_weekly) is still open if you haven't taken it yet, but probably not for long!
- The [JavaScript Advent Calendar](https://javascript.christmas/2020) rolls on - plenty to enjoy here.
