# JavaScript Weekly Issue 516: November 27, 2020

## Top Stories

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60/uw4wnneqstkas5par3ue.jpg)

[Flash Animations Live Forever at the Internet Archive](http://blog.archive.org/2020/11/19/flash-animations-live-forever-at-the-internet-archive/) — I know that many readers built Flash powered sites and used JavaScript’s *ActionScript* cousin before coming to JavaScript. If so, this may be a walk down memory lane. The Internet archive are now archiving and emulating Flash animations (using [Ruffle](https://ruffle.rs/), a Rust powered Flash emulator) in the browser thanks to WebAssembly.

Jason Scott

Josh W Comeau

[Optimizing Your Pages for the Back/Forward Cache](https://web.dev/bfcache/) — Firefox and Safari cache the state of entire pages (rather than just the underlying assets) so if you go back or forward, pages can be restored instantly. Chrome is gradually adopting similar features and there are some things you should take into account.

Philip Walton (Google)

[Use `console.log()` Like a Pro](https://denic.hashnode.dev/use-consolelog-like-a-pro) — I can already hear people arguing pros don’t use `console.log`, but the polls usually say otherwise ;-) Nonetheless, some nicely demonstrated tips here.
