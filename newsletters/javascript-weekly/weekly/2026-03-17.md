## Top Stories

[Temporal: The 9-Year Journey to Fix Time in JavaScript](https://bloomberg.github.io/js-blog/post/temporal/) — JavaScript’s date/time handling is notoriously [messy](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Temporal#background_and_concepts) and libraries like [Moment.js](https://momentjs.com/) became popular as a way to work around it. In 2017, Maggie Johnson-Pint, a maintainer of Moment.js, proposed the [Temporal API](https://tc39.es/proposal-temporal/docs/) to fix date/time handling for good, and we’re *mostly there* ([support is growing](https://caniuse.com/temporal), with Safari and Node to catch up).

Jason Williams (Bloomberg)

[Vite 8.0 Released](https://vite.dev/blog/announcing-vite8) — A mega release for the popular build tool. Designed to be a smooth upgrade, there’s a *lot* behind the scenes: `@vitejs/plugin-react` v6 no longer needs Babel, [Rolldown](https://rolldown.rs/) replaces Rollup and esbuild, Wasm SSR support, browser console forwarding to the terminal, and big performance gains.
