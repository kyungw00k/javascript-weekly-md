[Faster, Smarter JavaScript Debugging in Firefox DevTools](https://hacks.mozilla.org/2019/05/faster-smarter-javascript-debugging-in-firefox/) — If you haven’t tried out Firefox for a while, now’s a great time as the JavaScript debugging tools in the [developer edition](https://www.mozilla.org/en-US/firefox/developer/) have had some huge improvements, such as support for setting breakpoints *within* a line, [logpoints](https://developer.mozilla.org/en-US/docs/Tools/Debugger/Set_a_logpoint), and the ability to debug Web Workers.

[A Faster Script Loading Future with BinaryAST?](https://blog.cloudflare.com/binary-ast/) — BinaryAST is an over-the-wire format for JavaScript (backed by both Mozilla and Cloudflare) that aims to speed up parsing while keeping JavaScript’s semantics intact. Here’s how it works, what the benefits might be, and how Cloudflare are hoping to lean on BinaryAST in its goal to make the Web faster.

[What's New in V8 Release V7.5](https://v8.dev/blog/v8-release-75) — The latest branch of the V8 JavaScript engine has been unveiled, giving us the usual view of what’s coming to Node and Chrome soon.
