# JavaScript Weekly Issue 513: November 6, 2020

## Top Stories

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60/rdhgwdgz5zwn8tnugdza.jpg)

[What's the Deal with Svelte and SvelteKit?](https://svelte.dev/blog/whats-the-deal-with-sveltekit) — Two weeks ago we linked to [a video](https://www.youtube.com/watch?v=qSfdtmcZ4d0) where Svelte’s creator pondered the future of Web development. Now here’s a more definitive writeup of what you need to know about how Svelte is progressing.

Rich Harris (Svelte)

BTW, if [Svelte](https://svelte.dev/) is your thing, [Svelte Radio](https://www.svelteradio.com/episodes) is a frequent podcast covering developments in the space. There's also an official [Svelte November 2020 update.](https://svelte.dev/blog/whats-new-in-svelte-november-2020)

[WTFJS: A List of Funny and Tricky JavaScript Examples](https://github.com/denysdovhan/wtfjs#readme) — This has been around for a few years but continues to get tweaks and adjustments and if you’ve not seen it, it’s worth a look to get a deeper look into some of the more “uh, what?” corners of JavaScript.

Denys Dovhan

![](https://copm.s3.amazonaws.com/ea72763d.png)

[Vue 3 Is Out! Jump into Vue 3 Quickly in This New Course by Sarah Drasner](https://frontendmasters.com/courses/vue-3/?utm_source=newsletter&utm_medium=email&utm_campaign=javascriptweekly&utm_content=vue-3) — Learn fundamental concepts in Vue, such as directives, methods, and computed watchers. You’ll learn Vue’s reactivity system and the new Composition API in Vue 3.

Frontend Masters sponsor

[New, Core 'Redux Fundamentals' Tutorial](https://redux.js.org/tutorials/fundamentals/part-1-overview) — A complete rewrite of the official Redux introductory tutorial that covers both how Redux works and how to use it. Say goodbye to outdated terminology and hello to modern practices and code sandbox demos. Easy to skim even if Redux isn’t your cup of tea.

Redux Core Team

[Node v15.1.0 (Current) Released](https://nodejs.org/en/blog/release/v15.1.0/) — More bits and pieces to play around with compared to the typical minor Node release including diagnostics channels (a new experimental module for creating named channels to pass diaganostics messages around with), a new `spawn` event for child processes, and more.

Michaël Zasso

**⚡️ Quick bytes:**

- Hackaday has [profiled a neat TTL logic simulator](https://hackaday.com/2020/11/02/ttl-simulator-in-javascript/) built in JavaScript.
- The [latest VS Code release](https://code.visualstudio.com/updates/v1_51) has a new empty brace formatting option for JavaScript code (to set whether or not `{ }` should contain a space).
- [SpiderBasic](https://www.spiderbasic.com/) is a modern (and *commercial*) [BASIC](https://en.wikipedia.org/wiki/BASIC) implementation that compiles to JavaScript in an attempt to take BASIC to the Web.
- There's a **lot** of neat stuff in the *Code & Tools* section this week, so be sure to head down there! :-)

## 🛠 Code & Tools

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60/s1eciav1uplv499tlbpj.jpg)

[SpaceTime: A Lightweight Timezone Library](https://github.com/spencermountain/spacetime) — Use this to calculate times in other timezones. Has a Moment-esque API but is immutable. No dependencies. We last linked this years ago but it’s still under active development. It has [a neat interactive demo](https://observablehq.com/@spencermountain/spacetime) too.

Spencer Kelly

[Socket.io 3.0 Released: A Realtime Framework for Node and Browser](https://github.com/socketio/socket.io) — We’re awaiting a full blog post on this release, but Socket·io has been around many years and was once the ‘go to’ system for bidirectional browser/server communications that worked in almost every browser.

Socket·IO Project

[Square Terminal API](https://squareup.com/t/f_online/d_newsletter/p_javascriptweekly/l_us/dt_alldevice/pr_ecommerce_api/?route=docs/terminal-api/overview&site=developer.squareup.com) — Connect your app to the all-in-one payments device merchants love with a simple REST API. [Learn more](https://squareup.com/t/f_online/d_newsletter/p_javascriptweekly/l_us/dt_alldevice/pr_ecommerce_api/?route=docs/terminal-api/overview&site=developer.squareup.com).

[Dojo3D: A Library for Simple 3D Interactive Story Telling](https://github.com/webprofusion/dojo3d) — Aimed at ‘all ages of coder’, Dojo3D tries to make creating interactive 3D stories a smooth process. There’s [an online sandbox example](https://repl.it/@webprofusionchr/dojo3d-halloween) you can play with.

[gron: Make JSON Greppable](https://github.com/tomnomnom/gron) — A tool written in Go that transforms JSON into more easily `grep`pable assignments, so you can use `grep` and see the context/path of the result. Useful.

Tom Hudson

[Graphery SVG: A Library for SVG Creation and Management](https://www.graphery.org/svg/) — This [well-documented](https://www.graphery.org/svg/guide/) library is good for creating and manipulating SVG in JavaScript. Offers a simple API, that’s said to be very close to the native structure of SVG.
