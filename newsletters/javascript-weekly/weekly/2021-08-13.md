## Top Stories

[Vue 3.2 Released](https://blog.vuejs.org/posts/vue-3.2.html) — Some new features for single file components, significant perf improvements, and it’s now easy to define native custom elements with `defineCustomElement` (learn more about this in [Vue Components as Custom Elements](https://maximomussini.com/posts/vue-custom-elements/) – this could be a big deal for many).

Evan You

['JavaScript Needs More Helper Functions for Iteration'](https://2ality.com/2021/08/iteration-helpers.html) — Many languages have a plethora of functions or methods for iteration, but many of JavaScript’s options came late and scoped to particular data types (such as Array's new methods in ES6) and Axel thinks we need to go a lot further while comparing a variety of approaches.

Dr. Axel Rauschmayer

InfluxData sponsor

[V8 Release v9.3](https://v8.dev/blog/v8-release-93) — Recent V8 releases have been reasonably lean on new features, and so it goes with 9.3 which mostly gets faster compilation, `Object.hasOwn` (an alias for `Object.prototype.hasOwnProperty.call`), and the ability to attach error ‘causes’ to `Error` instances. It’s in beta until Chrome 93 (due in the coming weeks) and will appear in Node.js soon too.

Ingvar Stepanyan

[Next.js 11.1 Released](https://nextjs.org/blog/next-11-1) — The popular React framework takes a few steps forward with experimental ES modules support, a new integration with [swc](https://swc.rs/) to ultimately replace JS tooling (Babel and Terser), faster data mapping and source maps, and more.

Vercel
