# JavaScript Weekly Issue 543: June 18, 2021

## Top Stories

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60/zy1a5nqer7tplhf6zlou.jpg)

[Benchmarking JavaScript Memory Usage](https://blog.webpagetest.org/posts/benchmarking-javascript-memory-usage/) — Historically, memory usage hasn’t been a heavily tracked thing in the JavaScript world, whether for a lack of tooling or motivation. This is changing, and new tools and APIs are turning up all the time. Here, Tim has done some analysis of what the top 10k Web sites are using in terms of memory and, perhaps controversially, how the choice of framework can affect things.

Tim Kadlec (WebPageTest)

[Dr. Axel Explains Three ECMAScript Proposals](https://2ality.com/2021/06/error-cause.html) — JavaScript’s inimitable doctor is blogging again and has [examined the ‘error cause’ proposal](https://2ality.com/2021/06/error-cause.html) (a way to let `Error` instances specify if they were caused by another error), [JSON modules](https://2ality.com/2021/06/json-modules.html) (a way to import data as if it were an ES module), and [Accessible `Object.prototype.hasOwnProperty()`](https://2ality.com/2021/06/object-hasown.html) (a way to check if an object has an own/non-inherited property).

Dr. Axel Rauschmayer

Quick Bits

- [Gleam](https://gleam.run/) is a type-safe language for the Erlang virtual machine and it [can now compile to JavaScript](https://gleam.run/news/v0.16-gleam-compiles-to-javascript/) which opens up some new interesting frontend possibilities.
- GitHub's CEO has noted that [4 billion packages are downloaded each day](https://twitter.com/natfriedman/status/1405524163268288513) from the official npm registry.
- Did you know [you can use underscores](https://davidwalsh.name/numeric-separators) to make numbers easier to read? For example: `1_000_000`
- The Angular project has updated [its roadmap](https://angular.io/guide/roadmap) covering both in-progress improvements and future initiatives.
- V8 has added support for [import assertions](https://v8.dev/features/import-assertions).

Releases

[NeutralinoJS 2.3.0](https://github.com/neutralinojs/neutralinojs) — Cross-platform desktop app dev framework. [SVG.js 3.1](https://svgjs.dev/docs/3.1/) — Easier manipulation of SVG from JS. [ws 7.5.0](https://github.com/websockets/ws) — WebSocket client and server for Node.js. [Intro.js 4.1](https://introjs.com/) — Easy user onboarding approach. [Tailwind CSS 2.2](https://blog.tailwindcss.com/tailwindcss-2-2) — Popular CSS framework.

[Deno 1.11 Released](https://deno.com/blog/v1.11) — A variety of enhancements in the latest release of the alternative server-side V8-powered JavaScript runtime:

- The Deno project now supplies [official Docker images](https://hub.docker.com/r/denoland/deno).
- You can now abort `fetch` requests in a Web-compatible way.
- `deno lint` is now considered stable.
- `BroadcastChannel` brings support for broadcasting messages amongst Web Workers.

Deno Core Team
