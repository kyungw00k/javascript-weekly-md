# JavaScript Weekly Issue 662: November 9, 2023

## Top Stories

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/lebvxxoaxur758edwwud.jpg)

[Reintroducing Angular with Angular v17](https://blog.angular.io/introducing-angular-v17-4d7033312e4b) — Angular first appeared in the shape of *AngularJS* in 2010 and helped launch a wave of large-scale JavaScript frameworks. Angular remains popular in many use cases but is often overlooked in favor of newer options. v17 takes a leap forward in both features *and* vision, with the team rebranding Angular and repositioning it as a modern solution:

- [Angular.dev](https://angular.dev/) is an all-new docs site and home for the project ([the new guides](https://angular.dev/overview) look fantastic). It's in beta till Angular v18 is released – you can [learn more about it here](https://blog.angular.io/announcing-angular-dev-1e1205fa3039).
- Hydration is now production ready.
- Vite and esbuild are the default for new projects.
- Improved support for creating server side apps.
- New, improved built-in control flow to make code less verbose.
- Google pulled out all the stops for [▶️ the 'Special Angular Event'](https://youtu.be/Wq6GpTZ7AX0?t=113) – an hour of talks, interviews and discussions to bring you up to speed.

Minko Gechev and the Angular Team
