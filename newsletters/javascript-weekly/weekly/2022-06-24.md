[Ecma International Approves ECMAScript 2022, But What’s New?](https://2ality.com/2022/06/ecmascript-2022.html) — Thanks Ecma – [ECMAScript 2022 is now an *approved* standard.](https://www.ecma-international.org/news/ecma-international-approves-new-standards-6/) JavaScript’s resident doctor brings us up to speed with what this means in practice, as well as answering a few common questions. You’ll already be using some of these features – this is mostly about tying up the formalities. On to ES2023..

Dr. Axel Rauschmayer

❓ If you're unfamiliar with ECMAScript as a term, it's the name of the generic, standardized language behind JavaScript implementations, as JavaScript is technically a trademark of the Oracle Corporation. [Longer explanation.](https://exploringjs.com/impatient-js/ch_history.html#standardizing-javascript)
