[Remix 3 Enters Beta — It's No Longer a React Framework](https://javascriptweekly.com/link/184764/web) — Remix has quite the back story. Created by the duo behind React Router in 2020 and seen as an [alternative to Next.js](https://javascriptweekly.com/link/184765/web), Remix was [acquired by Shopify](https://javascriptweekly.com/link/184766/web) in 2022 and its core ideas folded into [React Router v7](https://javascriptweekly.com/link/184767/web) in 2024. Now, a new direction: a full-stack, web standards-first framework with its own UI component model and… no React.

Michael Jackson (Remix)

## 📢 Elsewhere in the ecosystem

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/wv3jtbpvschb5vcdrhw3.jpg)

- How can you *not* love a project homepage where you're a cat in a convertible driving through an endless barrage of obstacles? [Crashcat](https://javascriptweekly.com/link/184746/web) is a JavaScript 3D rigid body physics library built for games, simulations, and web experiences, complete with [numerous fun examples.](https://javascriptweekly.com/link/184747/web)
- ✉️ Cloudflare has [open sourced *Agentic Inbox*](https://javascriptweekly.com/link/184748/web), a self-hosted React 19 and React Router 7-based web email app that ties together and heavily leans on numerous Cloudflare APIs.
- [Ladybird](https://javascriptweekly.com/link/184749/web) is a *"truly independent web browser"* with its renderer and JS engine built entirely from *scratch*, with an alpha release due later this year. In [the project's latest update](https://javascriptweekly.com/link/184750/web) they cover recent significant JS and CSS improvements.
- Tired of `localhost:3000` on your projects? Vercel's [Portless](https://javascriptweekly.com/link/184751/web) lets you run local dev servers using a more user-friendly `.localhost` hostname over HTTPS.
- [Thales](https://javascriptweekly.com/link/184761/web) is a TypeScript to [Lean](https://javascriptweekly.com/link/184762/web) compiler that type-checks a subset of TypeScript and emits a Lean sidecar, turning your code into a Lean module you can reason about.
