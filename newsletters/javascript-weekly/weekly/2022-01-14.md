[Parcel CSS: A New CSS Parser, Compiler, and Minifier Written in Rust](https://parceljs.org/blog/parcel-css/) — Continuing the trend of writing JS infrastructure in other languages, the [Parcel bundler](https://parceljs.org/) gains a new CSS compiler and minifier written in Rust. [Parcel 2.2](https://github.com/parcel-bundler/parcel/releases/tag/v2.2.0) has just come out and supports a new experimental CSS transformer plugin if you want to try it out, but you can use it standalone too.

Devon Govett

**IN BRIEF:**

- The always popular [State of JavaScript survey](https://stateofjs.com/) is back again for a joint 2021/22 run. The results are usually interesting so please take it if you get the chance.
- Nicholas C. Zakas has a new book about [Understanding JavaScript Promises](https://ebooks.humanwhocodes.com/promises) out. *(Note: It's email walled, but I've had a skim and it's worth it.)*
- Long term support for AngularJS (not to be confused with *Angular*) has [now ended.](https://blog.angular.io/discontinued-long-term-support-for-angularjs-cc066b82e65a) AngularJS isn't going away as such but you are strongly encouraged to upgrade to [Angular proper](https://angular.io/).
- If you're looking to dabble with Jamstack this year, then there's a free online conference [taking place later this month](https://cfe.dev/events/the-jam-2022/) that's worth a look.
- The folks behind Astro and Skypack have raised $7m and [formed The Astro Technology Company.](https://astro.build/blog/the-astro-technology-company/) Congratulations and we look forward to seeing what they achieve.
- The Angular project has unveiled [the RFC for Typed Forms.](https://blog.angular.io/typed-forms-the-rfc-is-here-53263e792c89)
