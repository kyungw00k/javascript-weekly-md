Quick Bits

- The Vue project has decided to [not support IE11 with Vue 3](https://github.com/vuejs/rfcs/blob/ie11/active-rfcs/0000-vue3-ie11-support.md) — instead, numerous features will be backported to Vue 2.
- Node saw a [swathe of security releases](https://nodejs.org/en/blog/vulnerability/april-2021-security-releases/) this week to fix a few high severity issues, mostly in the OpenSSL dependency.
- The Chrome DevTools team have continued to write about how they're migrating the tools [over to using TypeScript.](https://developer.chrome.com/blog/migrating-to-typescript/) They've previously written similarly [about Puppeteer too](https://developer.chrome.com/blog/puppeteer-typescript/).
