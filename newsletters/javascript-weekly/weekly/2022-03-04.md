# JavaScript Weekly Issue 579: March 4, 2022

## Top Stories

[TypeScript 4.6 Released (And It Even Benefits Non-TS Users!)](https://devblogs.microsoft.com/typescript/announcing-typescript-4-6/) — The typed superset of JavaScript takes another step forward with the addition of the `es2022` target, allowing code in constructors *before* `super()`, improved recursion depth checks, more syntax errors that can be detected in JavaScript files if using VS Code (even if you're not developing in TypeScript yourself!), and more.

Daniel Rosenwasser

[Five Things You *Don't* Need JavaScript for](https://lexoral.com/blog/you-dont-need-js/) — We love JS, but CSS and SVG have progressed enough to replace some traditional JS use cases.

[Socket: See Potential Security Issues for `npm` Packages](https://socket.dev/blog/introducing-socket) — An interesting new project that scans the code of each npm package in an attempt to characterize their behavior, which is then reported on project specific pages, such as [this one for lodash](https://socket.dev/npm/package/lodash) or [this one for zx.](https://socket.dev/npm/package/zx) Early days but some potentially useful info here.

Socket

[🇺🇦 Leaflet Project Appeals to JS Developers for Ukraine](https://leafletjs.com/index.html) — Did you know the popular [Leaflet](https://leafletjs.com/SlavaUkraini/) mapping library was created by a Ukrainian [developer](https://twitter.com/mourner)? The Leaflet team has a message to share about the current invasion and how you can help. We, too, have donated and will continue to do so. In related news, GitHub has also shared [their response to the war](https://github.blog/2022-03-02-our-response-to-the-war-in-ukraine/).

Leaflet.js

**IN BRIEF:**

- Another month, [another VS Code release.](https://code.visualstudio.com/updates/v1_65) This time we get improved automatic language detection, lazy variable evaluation when debugging JS (and JS only, for now), and improved syntax error reporting in JS files.
- 📆 [DevOps.js](https://devopsjsconf.com/badge/jared_palmer?from=javascriptweekly) is a virtual conference on building, deploying, and monitoring JS apps taking place in just three weeks (March 24-25). [You can register here.](https://devopsjsconf.com/badge/jared_palmer?from=javascriptweekly)
- Apple, Bocoup, Google, Mozilla and others have announced [Interop 2022](https://webkit.org/blog/12288/working-together-on-interop-2022/), a collaboration to improve interoperability of (mostly visual, for now) Web technologies.
- The much loved and heavily used [MDN Web Docs](https://developer.mozilla.org/en-US/) site now has a new homepage, design, and logo.
