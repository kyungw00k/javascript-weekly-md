[Svelte 3 Released: Rethinking Reactivity](https://svelte.dev/blog/svelte-3-rethinking-reactivity) — [Svelte](https://svelte.dev/) is one of the most interesting UI frameworks out there as it’s not scared of taking a unique approach. Rather than running in the browser, Svelte runs at *build* time, compiling your app into more efficient runtime JavaScript. Svelte 3 takes more steps forward, particularly in helping you [write less code.](https://svelte.dev/blog/write-less-code)

Rich Harris

[Node.js 12 Released, And It's a Big One](https://medium.com/@nodejs/introducing-node-js-12-76c41a1b3f3f) — The newest ‘current’ branch of Node.js upgrades to [V8 7.4](https://v8.dev/blog/v8-release-74) which brings many memory and perf improvements for free as well as support for private class fields, TLS 1.3, diagnostic reports, [all-new ES module support](http://2ality.com/2019/04/nodejs-esm-impl.html), and more. NodeSource also have [a webinar on new features in Node 12](https://pages.nodesource.com/new-features-nodejs12-0501-jsw.html) next Wednesday.

Node.js Foundation
