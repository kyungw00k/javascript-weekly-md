# JavaScript Weekly Issue 632: March 31, 2023

## Top Stories

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/ng1pkebslgvvejvtw6oj.jpg)

[JavaScript Import Maps Now Supported Cross-Browser](https://web.dev/import-maps-in-all-modern-browsers/) — ES modules provide a modern way to include and reuse JavaScript code in web apps, and import maps provide the bridge between using module names in code and where those modules can actually be loaded from.

Thomas Steiner (Chrome Developers)

🍏 The import map news comes off the back of [the release of Safari 16.4](https://webkit.org/blog/13966/webkit-features-in-safari-16-4/) which introduces a *lot* of new functionality to the Mac-based browser, from [Web Push](https://web.dev/push-notifications-in-all-modern-browsers/) and import maps to improved Web Component support and lazy-loading support for iframes.

[Updates from the 95th TC39 Meeting](https://dev.to/hemanth/updates-from-the-95th-tc39-meeting-ne5) — Working group meetings might not *seem* interesting on the surface, but a lot of what TC39 discusses ends up in our day to day code. Of special interest this time are the progression of [support for *ranges*](https://github.com/tc39/proposal-iterator.range) and the [async context proposal](https://github.com/tc39/proposal-async-context/) to stage 2, and [awaiting a dictionary of promises](https://github.com/tc39/proposal-await-dictionary) and [decorators for class constructors/methods](https://github.com/tc39/proposal-class-method-parameter-decorators) to stage 1.

Hemanth HM

![](https://copm.s3.amazonaws.com/41c80c8c.png)

[React Authentication, Simplified](https://userfront.com/tutorials/react-authentication?utm_source=javascriptweekly.com&utm_medium=newsletter&utm_campaign=primary&utm_content=react_authentication) — In this article, we lay out a new approach to authentication (plus access control & SSO) in React applications.

Userfront sponsor

[React Labs: What React Core Has Been Working On](https://react.dev/blog/2023/03/22/react-labs-what-we-have-been-working-on-march-2023) — The latest update from the React core team. The most striking update comes around the progress of *React Forget*, an optimizing compiler with the goal being able to build fully reactive systems using the JavaScript and React mental models you already have, with the compiler tackling the trickiest parts.

The React Core Team
