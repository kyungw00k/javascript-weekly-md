## Top Stories

[Bare: A New Lightweight Runtime for Modular JS Apps](https://bare.pears.com/) — Imagine something like Node.js but really stripped back: *bare*, if you will. Like Node, it’s built on top of V8 and libuv (though it's designed to support multiple JavaScript engines) but Bare’s approach is to provide as little as possible (a module system, addon system, and thread support) and then rely upon userland modules that can evolve independently of Bare itself. It’s an interesting idea – [more details here.](https://pears.com/news/introducing-bare-actually-run-javascript-everywhere/)

Holepunch

[An Update on the Deno v Oracle JavaScript™ Fight](https://deno.com/blog/deno-v-oracle3) — Deno filed a petition with the USPTO to [cancel the 'JavaScript' trademark](https://javascript.tm/), as claimed by Oracle, and Oracle stepped up to fight back. Ryan recaps the basic story and asks for help to get the word out ([signing the open letter](https://javascript.tm/) is a great start, if you agree Oracle abandoned the trademark).

Ryan Dahl

[React 19.1 Released](https://github.com/facebook/react/releases/tag/v19.1.0) — The headline feature is *Owner Stacks*, a dev-only feature to track which components are responsible for rendering other components. 19.1 also brings fixes, small additions (like support for streaming in edge environments), [a new API for prerendering RSCs on the server](https://github.com/facebook/react/pull/31724), and enhanced Suspense support.
