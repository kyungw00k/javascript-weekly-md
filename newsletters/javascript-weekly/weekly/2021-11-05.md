# JavaScript Weekly Issue 563: November 5, 2021

## Top Stories

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60/j9dse6c4bgs3xz9notdw.jpg)

[Chrome's New Feature to Record and Replay User Flows](https://developer.chrome.com/docs/devtools/recorder/) — It’s only a preview feature in Canary builds right now, but Chrome is adding a new *Recorder* panel to its DevTools which lets you record and replay a string of actions on a page (including under simulated slow network conditions). You can also measure the flow and get a performance trace for analysis.

Jecelyn Yeen (Google)

[Angular 13 Released](https://blog.angular.io/angular-v13-is-now-available-cce66f7bc296) — Despite sentiment on social media, Angular continues to be a hugely popular framework, particularly in enterprise situations and v13 continues to push things forward with more of the [Ivy](https://developer.okta.com/blog/2020/02/12/angular-ivy) related improvements coming through.

**QUICK BITS:**

- The V8 team has created [the V8 v9.7 branch](https://v8.dev/blog/v8-release-97) which introduces two new Array methods (`findLast` and `findLastIndex`) without needing a flag.
- Two more npm packages have [had malicious code injected](https://www.bleepingcomputer.com/news/security/popular-coa-npm-library-hijacked-to-steal-user-passwords/) by hijackers, this time it's `coa` and `rc`, both of which get millions of downloads per week.
- Jeremy Wagner has written and now published, with *A Book Apart*, [a new book called *Responsible JavaScript*](https://jeremy.codes/blog/i-went-and-wrote-a-book/).
- Nat Friedman, who has been a fantastic steward and CEO for GitHub since its acquisition by Microsoft, [is moving on](https://github.blog/2021-11-03-thank-you-github/) and handing the reins [over to Thomas Dohmke](https://github.blog/2021-11-03-building-the-next-phase-of-github-together/).
- Talking of GitHub, if you fancy building a game in JavaScript consider joining their [Game Off 2021 event](https://github.blog/2021-11-01-game-off-2021-theme-announcement/) – you get till December 1 to submit an entry.

[Hasty: A JavaScript Snippet Perfomance Comparison Tool](https://hasty.dev/) — A quick, easy-to-use online tool that lets you test variations of a particular snippet to see which one performs best. A bit like *jsPerf* from the old day, or other tools like [perf.link](https://perf.link/) and [jsBench](http://jsbench.github.io/).

[Coding Font: A 'Game' to Find Your New Favorite Code Font](https://www.codingfont.com/) — This is a fun idea. It throws over 25 popular developer-oriented fonts into a ‘tournament’ and you then pick your favorites until you’re left with a winner. (It was *JetBrains Mono* for me.)
