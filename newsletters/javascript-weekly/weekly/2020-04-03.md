[ECMAScript 2020: The Final Feature Set](https://2ality.com/2019/12/ecmascript-2020.html) — TC39 has just approved the [ECMAScript 2020 spec](https://tc39.es/ecma262/2020/) *(a full weekend's bedtime reading right there!)* with Ecma GA approval due in a few months, but what’s new? Dr. Axel rounds it up with links to the included stage 4 proposals. If you prefer something more code-driven, Pawel Grzybek has [a similar roundup.](https://pawelgrzybek.com/whats-new-in-ecmascript-2020/)

Dr. Axel Rauschmayer

[TypeScript 3.9 Beta Released](https://devblogs.microsoft.com/typescript/announcing-typescript-3-9-beta/) — 3.9’s focus is on “performance, polish, and stability” with the most significant change you’re likely to notice being faster compile times.

Daniel Rosenwasser (Microsoft)

[Building with Web Components](https://blog.heroku.com/building-with-web-components) — Web components seek to tilt the balance of web development back towards a standard that has been agreed upon by browser vendors and developers. Explore the history and current state of web components in front-end development.

Heroku sponsor

[Perflink: Low Friction JS Benchmarks You Can Share via URL](https://perf.link/) — Version 2 just launched, [details here](https://twitter.com/lukejacksonn/status/1245386330978373633). Tests now run isolated in web workers, ability to save and fork to/from local storage, works cross-browser, and more.

luke jackson

[Rendering Charts with OffscreenCanvas](https://blog.scottlogic.com/2020/03/19/offscreen-canvas.html) — Rendering charts can be an intensive operation for browsers, but what if you could delegate such work to a Web Worker? This post looks at what performance improvements you can realistically expect.

Chris Price
