[Bun: An Exciting, Speedy New JavaScript Runtime](https://bun.sh/) — Tired of Node? Tired of Deno? Never. But a new challenger *has* appeared. Bun is aimed at 'outside of browser' use cases and is built around WebKit's *JavaScriptCore*, rather than V8. Some notable features:

- Includes its own bundler, transpiler, and task runner.
- Promising [performance](https://twitter.com/bwasti/status/1544690279650918400).
- Supports a lot of Node and Web APIs already.
- You can use npm packages with it.
- Works on macOS (x86 and Arm), Linux, or WSL.
