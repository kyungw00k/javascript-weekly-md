# JavaScript Weekly Issue 526: February 19, 2021

## Top Stories

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60/gpxzhuh4pg1yynxywrt9.jpg)

[Announcing Vite 2.0](https://dev.to/yyx990803/announcing-vite-2-0-2f0a) — [Vite](https://vitejs.dev/) is a build tool aimed at giving you a faster development experience by providing a dev server with hot module replacement support, and Rollup- and esbuild-powered bundling and building.

Evan You

[V8's Super Fast `super` Property Access](https://v8.dev/blog/fast-super) — The first of two articles this week about how the V8 team are somehow making the already fast V8 JavaScript engine even faster for us. Using `super`, till now, was an unoptimized and slightly costly operation but now it’s almost on par with accessing normal properties.

Marja Hölttä (V8)

![](https://copm.s3.amazonaws.com/cfe815b3.png)

[Frontity: Bringing React to WordPress](https://frontity.org/?utm_source=javascript-weekly&utm_medium=newsletter&utm_campaign=partnerships) — Frontity Framework connects with WordPress seamlessly so you can focus on building your React front-end, without worrying about tooling and configuration. Its ease of use and extensibility make it a favorite when working with headless WordPress.

Frontity sponsor

[An Interview with Ryan Dahl, the Creator of Node.js and Deno](https://evrone.com/ryan-dahl-interview) — Ryan created Node.js 11 years ago and in recent years has moved on to [Deno](https://deno.land/) where he’s attempting a new take at the same idea. This interview touches on Deno’s challenges and Ryan’s love of Rust and Vim.

Evrone

[V8's Route to Faster JavaScript Method Calls](https://v8.dev/blog/adaptor-frame) — This gets *very technical* and fast, but it’s yet more of a look behind what the V8 team is up to. This work improves the performance of your Node and browser-facing code alike.

Victor Gomes

[Opal 1.1: A Ruby to JavaScript Compiler](https://opalrb.com/) — An interesting way to take Ruby code to the frontend by simply translating it into JavaScript. It’s been around for years but [1.1.0](https://github.com/opal/opal/releases/tag/v1.1.0) is the first significant release in some time.

Adam Beynon and Contributors

Quick Bits

- [NASA used Three.js](https://twitter.com/mrdoob/status/1362508150507790343) to render a real-time simulation of this week's NASA rover landing on Mars 🚀
- The [date-fns documentation site](https://date-fns.org/docs/Getting-Started) lets you play with the library in the browser console in a cute way and [here's the code](https://github.com/date-fns/date-fns.org/blob/main/src/assets/template.ejs#L74) that makes it work.
- One of the Angular core team has written about [how Angular strongly encourages best practices](https://blog.angular.io/with-best-practices-from-the-start-d64881a16de8) and the next natural step is for Angular 12 to enable ['strict mode'](https://blog.angular.io/angular-cli-strict-mode-c94ba5965f63) by default.
- Node.js has some security releases [coming out next week](https://nodejs.org/en/blog/vulnerability/february-2021-security-releases/) – one of the security issues involved is of 'critical' severity. We will share these in next week's issue of [Node Weekly](https://nodeweekly.com/).
- [esbuild now supports](https://twitter.com/evanwallace/status/1362224191546613769) Apple's M1 chip natively and is *even faster*.
- The [Colyseus](https://www.colyseus.io/) open source Node-powered multiplayer game engine has been 'acquired' by Lucid Sight, a developer of popular licensed mobile games.
