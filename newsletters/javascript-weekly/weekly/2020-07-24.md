[Several New Features Promoted to Stage 4 at TC39](https://dev.to/hemanth/stage-4-features-5a26) — If you’re interested in the future of JavaScript, several features have been promoted to *stage 4* which, in [TC39 parlance](https://tc39.es/process-document/), means they are ‘finished’ and ready for inclusion in the formal ECMAScript standard. They include:

- `Promise.any` and `AggregateError`
- Number separator support (e.g. `1_000_000`)
- [Weak references](https://v8.dev/features/weak-references)
- Improvements to `Intl.DateTimeFormat`
- ..and more, naturally 😄

Hemanth HM

[Vue 3 Now in Release Candidate Stage](https://github.com/vuejs/rfcs/issues/189) — The final Vue 3.0 release isn’t due till sometime next month, but the API and implementation are now considered ‘stabilized’ with no new major features or breaking changes expected. There’s also [a beta of the devtools](https://github.com/vuejs/vue-devtools/releases/tag/v6.0.0-beta.1) with 3.0 support and a new [v3 focused documentation site.](https://v3.vuejs.org/) Also here's [some of what's new in Vue 3](https://www.grapecity.com/blogs/whats-new-in-vue-3).
