[React 17.0 Released](https://reactjs.org/blog/2020/10/20/react-v17.html) — The focus in React 17 is peace, harmony, and gentle progression, with [subtle](https://reactjs.org/blog/2020/09/22/introducing-the-new-jsx-transform.html) changes, making apps easier to upgrade gradually in future, and also to make it easier to embed React apps into apps built with other technologies.

Dan Abramov and Rachel Nabors

[Node 15 Released](https://medium.com/@nodejs/node-js-v15-0-0-is-here-deb00750f278) — The new ‘release’ line (the one that gets all the new features first) of Node is here. Two key features:

- An upgrade to the V8 8.6 engine (from 8.4) adding various JS features like `Promise.any()`, logical assignment operators, and `String.prototype.replaceAll()`
- Unhandled rejections are now raised as exceptions by default. If there's any one change that'll cause you some head scratching, it's this.

For more, check [this week's Node Weekly](https://nodeweekly.com/issues/361) ;-)

**⚡️ Quick bytes:**

- Want to see something cool? [MONOSPACE](http://www.p01.org/MONOSPACE/) is a demo (in the '[demoscene](https://en.wikipedia.org/wiki/Demoscene)' sense) written in 1021 bytes of JavaScript and it won the 1024 byte demo competition at Assembly 2020.
- Rich Harris (of [Svelte](https://svelte.dev/) fame) has [shared a video](https://www.youtube.com/watch?v=qSfdtmcZ4d0) exploring his thoughts on the future of Web development and where the [Sapper](https://sapper.svelte.dev/) Svelte-based framework slots in (or doesn't!) — Cool ideas here and a must watch if you're sick of endless bundling.
- [Vue 3.0.2](https://github.com/vuejs/vue-next/blob/master/CHANGELOG.md) is out – almost entirely bug fixes. Or how about [Ember 3.22](https://blog.emberjs.com/2020/10/20/ember-3-22-released.html) or the [first RC](https://github.com/angular/angular/releases/tag/11.0.0-rc.0) of Angular 11?
- Someone has noted that using `const` instead of `var` or `let` can cause [big slowdowns](https://github.com/evanw/esbuild/issues/478) in the JavaScriptCore engine Webkit (and Safari) uses. They're now on the case to resolve it.
- Most of this is double dutch to me but apparently Agoric lets you [write 'smart contracts' in JavaScript](https://decrypt.co/45758/javascript-smart-contract-project-taps-chainlink-speedy-defi-data) and a TC39 member is involved, if that's your thing. *(Note to self: learn about this stuff.)*
- We announced [webpack 5](https://webpack.js.org/blog/2020-10-10-webpack-5-release/) last week, but we're already getting [webpack 5.2.0](https://github.com/webpack/webpack/releases/tag/v5.2.0) this week.
