# JavaScript Weekly Issue 663: November 16, 2023

## Top Stories

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/idn5i3zweygdrffmarsa.jpg)

[Prettier 3.1 Released](https://prettier.io/blog/2023/11/13/3.1.0) — The popular opinionated code formatter has a new release including support for the new control flow syntax in Angular 17 plus a new, experimental formatting option for ternary expressions (as in `x ? y : z`) explained in more depth in ["A curious case of the ternaries."](https://prettier.io/blog/2023/11/13/curious-ternaries.html)

Alex Rattray

🤔 With an aim to making code formatting even faster, Prettier's co-creator has [put up a $10k bounty](https://twitter.com/Vjeux/status/1722733472522142022) for any Rust project that passes >95% of Prettier's tests.

[Exploring Explicit Resource Management in JS](https://iliazeus.github.io/articles/js-explicit-resource-management-en/) — A practical look at the idea of explicit resource management, an idea both [currently at stage 3 in TC39](https://github.com/tc39/proposal-explicit-resource-management) (and partly [supported in TypeScript 5.2](https://devblogs.microsoft.com/typescript/announcing-typescript-5-2/#using-declarations-and-explicit-resource-management)) that makes it easier to ‘clean up’ resources once they are no longer used or needed.

## 🛠 Code & Tools

[gsplat.js: A Gaussian Splatting Library](https://github.com/dylanebert/gsplat.js) — Gaussian splatting is an increasingly popular graphics rendering techniques where rather than render millions of tiny, textured triangles in a scene, you get a more wild paintball-like splatter fest, where each paintball creates a smooth, colorful blob instead of a rigid shape. [This demo](https://jsfiddle.net/e52x4m9z/) is both simple and striking.

Dylan Ebert

[Reveal.js 5.0: The HTML Presentation Framework](https://github.com/hakimel/reveal.js/releases/tag/5.0.0) — A way to build presentations using standard Web technologies. The [project’s homepage](https://revealjs.com/) is, itself, such a presentation. v5.0 adds support for ‘scroll mode’ which essentially turns presentations into more typically scrolling Web experiences – [demo.](https://slides.com/news/scroll-mode/scroll)

[Datasheet Grid: An Airtable-like React Component](https://react-datasheet-grid.netlify.app/) — If you’ve got an array of objects and you want a way for users to manipulate them, this is for you.
