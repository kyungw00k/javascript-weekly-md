[Introducing Crank.js: A New JSX-Driven Framework](https://crank.js.org/blog/introducing-crank) — A new framework for creating JSX-driven components with functions, promises and generators. If you haven’t got time to read the creator’s lengthy (but well worth reading!) rationale for creating another framework, [check out the GitHub repo](https://github.com/bikeshaving/crank) or jump straight into [a CodeSandbox example.](https://codesandbox.io/s/a-loading-crank-component-pci9d) I hope to see a lot more about this in the near future.

Brian Kim

[The Cost of JavaScript Frameworks](https://timkadlec.com/remembers/2020-04-21-the-cost-of-javascript-frameworks/) — Using data from the [HTTP Archive](https://httparchive.org), Tim explores the performance cost of using various popular frameworks as a starting point. Some good analysis here.

Tim Kadlec

[Node.js 14 Released](https://medium.com/@nodejs/node-js-version-14-available-now-8170d384567e) — v14 now becomes the current ‘release’ line with it becoming a LTS (Long Term Support) release in October. [Diagnostic reports](https://medium.com/the-node-js-collection/easily-identify-problems-in-node-js-applications-with-diagnostic-report-dc82370d8029) are now a stable feature, it now uses [V8 8.1](https://v8.dev/blog/v8-release-81), there's an experimental [Async Local Storage API](https://nodejs.org/api/async_hooks.html#async_hooks_class_asynclocalstorage), and more.

Michael Dawson and Bethany Griggs

[Vidact: A Compiler to Convert React-like Apps to Vanilla JS](https://mohebifar.github.io/vidact/) — Compiles pure React-compatible JavaScript (JSX) to vanilla JavaScript with no virtual DOM.. think similar to Svelte, but without introducing a new syntax. A curious idea.

Mohamad Mohebifar

[Highlight.js 10.0: The Web Syntax Highlighter Library](https://highlightjs.org/) — I always laugh when I see a big release of a popular library and saw we last linked it [5 years ago](https://javascriptweekly.com/issues/221) 😄 Nonetheless, this *is* [a big release](https://github.com/highlightjs/highlight.js/releases/tag/10.0.0) and Highlight.js continues to be a fantastic library for syntax highlighting code.

highlight.js

[Creating Web Apps via TypeScript and webpack](https://2ality.com/2020/04/webpack-typescript.html) — The example app doesn’t use any front-end framework, just vanilla DOM APIs. Some knowledge of TypeScript, webpack, and npm are required, but this is a neat way to bring together a bunch of tools.

[What Is The `toJSON()` Function?](http://thecodebarbarian.com/what-is-the-tojson-function-in-javascript.html) — If an object has a `toJSON` function, `JSON.stringify()` calls `toJSON()` and serializes the return value from `toJSON()` instead.

Valeri Karpov
