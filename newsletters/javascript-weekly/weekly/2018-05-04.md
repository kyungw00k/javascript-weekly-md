[Read on the Web](https://javascriptweekly.com/issues/384)

[Angular 6.0 Released](https://blog.angular.io/version-6-of-angular-now-available-cc56b0efa7a4) — A major release focused on Angular’s toolchain (particularly Angular CLI) and developer experience than on the framework itself. *Angular Elements* also allows you to bootstrap components within an existing app by registering them as Custom Elements. Want help in upgrading to 6 from an earlier version? Use [the Angular Update Guide.](https://update.angular.io/)

Stephen Fluin

[Dojo 2.0: A Step Forward for an Early JS Framework](https://dojo.io/blog/2018/05/02/2018-05-02-Dojo2-0-0-release/) — Originally known as the *Dojo Toolkit*, Dojo pioneered a lot of concepts we now take for granted. Dojo 2 brings modern standards to the framework such as a reactive, virtual DOM-based widget system, Web Components support, and CLI tooling.

The JS Foundation
