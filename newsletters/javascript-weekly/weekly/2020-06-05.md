**⚡️ Quick bytes:**

- 🖤 This week, the React core team [joined the Facebook employee walkout](https://twitter.com/dan_abramov/status/1267544361929256966) in solidarity with the Black community — it's [time to speak up](https://auth0.com/blog/its-time-to-speak-up/), [donate](https://pitchfork.com/thepitch/resources-in-the-fight-against-police-brutality-and-systemic-racism/) or even just [stream a fundraising video](https://www.youtube.com/watch?v=bCgLa25fDHM), because Black lives matter.
- The Repl.IT online IDE/sandbox now [supports Deno](https://repl.it/languages/deno) if you want to play without installing it for real.
- SpiderMonkey, Firefox's JavaScript engine, has [a new regular expressions engine](https://hacks.mozilla.org/2020/06/a-new-regexp-engine-in-spidermonkey/) which opens up support for modern regex syntax it didn't have before.
- [AngularJS 1.8.0](https://github.com/angular/angular.js/releases/tag/v1.8.0) is out. Yes, that's the old school Angular.js, not the more modern [Angular.](https://angular.io/)
- Zappar has released some [universal AR SDKs](https://www.zappar.com/blog/introducing-universal-ar-sdks-threejs-unity-frame-more/) for Three.js and vanilla JavaScript if augmented reality is your bag.
