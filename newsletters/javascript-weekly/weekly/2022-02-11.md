## 🛠 Code & Tools

[zx 5.0: Writing Better Shell Scripts in JavaScript](https://github.com/google/zx) — Google isn’t slacking, with the fourth major version of `zx` in a year. Rather than using something like `bash` to throw together a quick script, `zx` provides niceties to do the same with JavaScript. [v5.0](https://github.com/google/zx/releases/tag/5.0.0) adds built-in YAML support.

Google

[Pintora: An Extensible Text-to-Diagram Rendering Library](https://github.com/hikerpig/pintora) — A similar idea to the well established [Mermaid.js](https://mermaid-js.github.io/mermaid/#/) (which has [a new version](https://github.com/mermaid-js/mermaid/releases/tag/8.14.0) out today, too) but with a different attitude to extensibility as well as no requirement for a headless browser if you're using it from Node. The [introduction docs](https://pintorajs.vercel.app/docs/intro/) have both visual and code examples.
