# JavaScript Weekly Issue 710: October 24, 2024

## Top Stories

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/nzdnecxa3eqexjde35fv.jpg)

[Svelte 5 is Alive](https://svelte.dev/blog/svelte-5-is-alive) — The long awaited next major release of [Svelte](https://svelte.dev/), the compiler-driven JS UI framework, is the *“most significant release in the project’s history”*, while remaining largely backwards compatible. A big addition is *runes* for explicitly declaring reactive state, but there’s [much more besides.](https://vercel.com/blog/whats-new-in-svelte-5) The official [svelte.dev](https://svelte.dev/) site has also [undergone a big rebuild](https://svelte.dev/blog/the-omnisite) to act as an ‘omnisite’ for all things Svelte.

The Svelte Team

📺 If you want to see *how* to use Svelte 5, *Syntax*'s Scott Tolinski has released [▶️ a 2 hour Svelte 5 basics course](https://www.youtube.com/watch?v=8DQailPy3q8) on YouTube.

[FlexGrid by Wijmo: The Industry-Leading JavaScript Datagrid](https://developer.mescius.com/wijmo/flexgrid-javascript-data-grid?utm_source=CooperPress&utm_medium=JavaScript-Weekly&utm_campaign=Wijmo-JS-Weekly-Primary-Sponsor-Oct-2024) — A fast and flexible DataGrid for building modern web apps. [GenAIScript: Microsoft's Generative AI Scripting Environment](https://microsoft.github.io/genaiscript/) — Microsoft is really poking at AI from all angles. [TypeChat](https://github.com/microsoft/TypeChat) introduced a type-safe way to talk to LLMs; now *GenAIScript* offers a JavaScript-powered way to programmatically assemble prompts and process responses. They claim it *“brings essential LLM prompt tooling into a cohesive scripting environment.”*

Microsoft

[Next.js 15 Released](https://nextjs.org/blog/next-15) — It’s a big week for the popular (some may even argue *default*) React framework with [Next.js Conf](https://nextjs.org/conf) starting today, as well as this release. It includes a codemod CLI for easier upgrading, async request APIs, alignment with React 19, and more.

Vercel

**IN BRIEF:**

- Developer IDE firm *JetBrains* has announced its [WebStorm JavaScript/TypeScript IDE](https://www.jetbrains.com/webstorm/) is [now free for non-commercial use](https://blog.jetbrains.com/blog/2024/10/24/webstorm-and-rider-are-now-free-for-non-commercial-use/) which, notably, also covers paid content creators.
- The popular `shadcn/ui` component library has [unveiled a new suite of components for building *sidebars.*](https://ui.shadcn.com/docs/components/sidebar)
- ⌚ [Spectra](https://www.kickstarter.com/projects/pocuter/spectra) is a hackable smartwatch seeking funding on Kickstarter (but it has passed its initial goal). ESP32-S3 based, it appears able to run JavaScript as shown in the 'dev experience' screenshots.
- 🙋 This year's [*State of React* survey](https://survey.devographics.com/en-US/survey/state-of-react/2024) is now open to take until November 19.

**RELEASES:**

- ⚛️ [React Native 0.76](https://reactnative.dev/blog/2024/10/23/release-0.76-new-architecture) – A *big* release. The so-called ['New Architecture' is now used by default](https://reactnative.dev/blog/2024/10/23/the-new-architecture-is-here) (so you can now use all the modern React features in RN too), the React Native DevTools are stable, and build times are faster than ever.
- 🥖 [Bun](https://bun.sh/) is on a *roll..* with three releases in the past week. [v1.1.31](https://bun.sh/blog/bun-v1.1.31) added support for `node:http2` server and gRPC, [v1.1.32](https://bun.sh/blog/bun-v1.1.32) added Node's `crypto.hash` method, and [v1.1.33](https://bun.sh/blog/bun-v1.1.33) was a bug-fix release.
- [Express.js 5.0](https://expressjs.com/2024/10/15/v5-release.html) – Landed a few weeks ago, but now there's an official post explaining the state of the project which has now [passed a comprehensive third party security audit.](https://expressjs.com/2024/10/22/security-audit-milestone-achievement.html)
- [Medusa 2.0](https://medusajs.com/blog/v2-release/) – A Node.js-powered ecommerce platform.
- [React Compiler Beta](https://react.dev/blog/2024/10/21/react-compiler-beta-release), [Turborepo 2.2](https://turbo.build/blog/turbo-2-2-0), [ESLint v9.13.0](https://eslint.org/blog/2024/10/eslint-v9.13.0-released/), [Deno 2.0.2](https://github.com/denoland/deno/releases/tag/v2.0.2)
