# JavaScript Weekly Issue 625: February 10, 2023

## Top Stories

It looked quiet at first but wow, what an epic week this turned out to be. There's a lot to chew on here, and we even have a variety of *bonus* items at the very end of the issue. Enjoy! __ *Your editor, Peter Cooper*

[Speeding Up the JS Ecosystem: It's ESLint's Turn](https://marvinh.dev/blog/speeding-up-javascript-ecosystem-part-3/) — Last year we featured [an article](https://marvinh.dev/blog/speeding-up-javascript-ecosystem/) from the same author about how he was finding, and fixing, low-hanging performance fruit in popular JavaScript projects. They also talk about [Javy](https://github.com/Shopify/javy), a JS to WebAssembly toolchain being built at Shopify that lets you run JS code on a WASM-embedded JS runtime.
