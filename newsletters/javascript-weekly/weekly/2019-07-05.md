[Babel 7.5.0 Released](https://babeljs.io/blog/2019/07/03/7.5.0) — The popular JavaScript compiler now includes an official plugin for dynamic `import`s ([currently in stage 4](https://github.com/tc39/proposal-dynamic-import)) and the [F# style pipeline operator.](https://github.com/valtech-nyc/proposal-fsharp-pipelines/blob/master/README.md) There’s also *experimental* support for TypeScript namespaces.
