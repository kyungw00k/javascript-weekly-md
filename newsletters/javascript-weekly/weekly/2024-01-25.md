## Top Stories

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/zjtcrrcfcw1nclxr4hjg.jpg)

[Better JavaScript Shell Scripting with *Bun Shell*](https://bun.sh/blog/the-bun-shell) — Performance-focused server-side runtime [Bun](https://bun.sh/) continues its rapid evolution with a side quest into the world of shell scripting by making it easier, cross-platform friendly, and less verbose. [zx](https://github.com/google/zx) offers similar (but less integrated and focused) functionality to Node users, if you should prefer.

Jarred Sumner (Bun)

[Announcing AdonisJS v6](https://adonisjs.com/blog/adonisjs-v6-announcement) — If you want a backend Web framework with [amazing docs](https://docs.adonisjs.com/guides/introduction) and packed with features, [Adonis](https://adonisjs.com/) is a good choice. v6 is a big step forward moving to ESM by default, with more type safety overall (including for routes and middleware references), a new validation library, and more.

Harminder Virk

Frontend Masters sponsor

[QuickJS: The Small, Embeddable JavaScript Engine](https://bellard.org/quickjs/) — Several years ago, Fabrice Bellard, the genius behind FFMPEG and JSLinux, built a tiny and complete JavaScript engine in C. It now supports ES2023 and its latest release adds top-level `await` in modules and its REPL, as well as support for some cutting edge JS features ([changelog](https://bellard.org/quickjs/Changelog)).

Fabrice Bellard
