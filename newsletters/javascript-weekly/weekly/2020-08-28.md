ES6) internally, and now passes events directly to listeners. [Full list of changes.](https://github.com/d3/d3/blob/master/CHANGES.md) There’s also [a 5.x to 6.0 migration guide](https://observablehq.com/d/f91cccf0cad5e9cb) for existing users.

## 🔧 Code & Tools

[fastest-levenshtein: Performance Oriented Levenshtein Distance Implementation](https://github.com/ka-weihe/fastest-levenshtein) — Levenshtein distance is a metric for measuring the differences between two strings (usually). This claims to be *the fastest* JS implementation, but we’ll let benchmarks be the judge of that :-)

ka-weihe

[Yarn 2.2 (The Package Manager and `npm` Alternative) Released](https://dev.to/arcanis/yarn-2-2-dedupe-faster-lighter-ha5) — As well as being smaller and faster, a [`dedupe` command](https://yarnpkg.com/cli/dedupe) has been added to deduplicate dependencies with overlapping ranges.

Maël Nison

[Light Date ⏰: Fast and Lightweight Date Formatting for Node and Browser](https://github.com/xxczaki/light-date) — Comes in at 157 bytes, is well-tested, compliant with Unicode standards on dates, and written in TypeScript.

Antoni Kepinski

[Barebackups: Super-Simple Database Backups](https://barebackups.com) — We automatically backup your databases on a schedule. You can use our storage or bring your own S3 account for unlimited backup storage.

Barebackups sponsor

[Carbonium: A 1KB Library for Easy DOM Manipulation](https://github.com/edwinm/carbonium) — Edwin submitted this himself, so I’ll let him explain it in his own words: *“It’s for people who don’t want to use a JavaScript framework, but want more than native DOM. It might remind you of jQuery, but this library is only around one kilobyte and only supports native DOM functionality.”*

Edwin Martin

[DNJS: A JavaScript Subset for Configuration Languages](https://github.com/leontrolski/dnjs) — You might think that JSON can already work as a configuration language but this goes a step further by allowing various other JavaScript features in order to be more dynamic. [CUE](https://cuelang.org/) and [Dhall](https://dhall-lang.org/) are other compelling options in this space.

Oliver Russell

[FullCalendar: A Full Sized JavaScript Calendar Control](https://fullcalendar.io/) — An interesting option if you want a Google Calendar style control for your own apps. Has connectors for React, Vue and Angular. The base version is MIT licensed, but there’s a ‘premium’ version too. [v5.3.0](https://github.com/fullcalendar/fullcalendar/releases/tag/v5.3.0) just came out.

Adam Shaw

[file-type: Detect The File Type of a Buffer, Uint8Array, or ArrayBuffer](https://github.com/sindresorhus/file-type) — For example, give it the raw data from a PNG file, and it’ll tell you it’s a PNG file. Usable from both Node and browser.

Sindre Sorhus
