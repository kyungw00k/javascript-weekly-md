[Logging Activity with the Web Beacon API](https://www.smashingmagazine.com/2018/07/logging-activity-web-beacon-api/) — The Beacon API is a Web API ([supported](https://caniuse.com/#feat=beacon) by all major browsers) that provides an efficient way for data to be asynchronously sent from a page back to a server for logging purposes.

[Ionic 4 Beta Released: Build Native Apps with Web Tech](https://blog.ionicframework.com/announcing-the-ionic-4-beta/) — The beta release of Ionic 4, a framework for building native apps and PWAs with Web technology, has *just* landed. 4.0 marks the first version to completely embrace modern Web APIs such as Custom Elements, CSS Variables and Shadow DOM, plus it’s framework-agnostic at its core.
