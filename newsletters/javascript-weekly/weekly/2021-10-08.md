[How Kent C Dodds Built a Modern Website in 2021](https://kentcdodds.com/blog/how-i-built-a-modern-website-in-2021) — Kent is well known in the JavaScript space from [Testing JavaScript](https://testingjavascript.com/), [Epic React](https://epicreact.dev/) and his popular [webdev podcast](https://kentcdodds.com/chats/04), and he’s just completed a big effort to rebuild his entire Web presence using React, Remix, XState, Prisma, Express.js, and numerous other JavaScript tools and technologies. This is a good look at how it’s come together.

Kent C Dodds

[When a `WeakMap()` Actually Came in Handy](https://macarthur.me/posts/when-a-weakmap-came-in-handy) — The creator of [slide-element](https://github.com/alexmacarthur/slide-element), an element animation library, explains a situation where he found the often misunderstood [WeakMap](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/WeakMap) type came in handy.

[jQuery Maintainers Continue Modernization Initiative; Deprecate jQuery Mobile](https://blog.jquery.com/2021/10/07/jquery-maintainers-continue-modernization-initiative-with-deprecation-of-jquery-mobile/) — Given its age and remaining importance on the Web, it’s always great to see updates from the jQuery team. [jQuery UI 1.13.0](https://blog.jqueryui.com/2021/10/jquery-ui-1-13-0-released/) has been released, but the big news is jQuery Mobile is deprecated as of yesterday, and jQuery UI is now [in maintenance only mode](https://blog.jquery.com/2021/10/07/jquery-maintainers-update-and-transition-jquery-ui-as-part-of-overall-modernization-efforts/).
