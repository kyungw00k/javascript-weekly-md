**IN BRIEF:**

- ⭐️ Last week, Vue.js's creator gave [▶️ a 'State of the Union'-style address](https://www.youtube.com/watch?v=1ntuhMzAzU8) on the project (plus Vite) for Vue.js Amsterdam.
- 👾 [OneJS](https://onejs.com/intro) is an intriguing JavaScript-based scripting engine for building UIs in the Unity game engine. There's [some example code](https://github.com/DragonGround/OverwatchSample) of using it to create a UI on top of a game.
- Someone scraped 1 million top domains to analyze what JS libraries they're relying upon. [There's a lot of data](https://github.com/get-set-fetch/scraper/tree/main/datasets/javascript-libs-from-top-1mm-sites) or just enjoy [this summary graphic](https://github.com/get-set-fetch/scraper/blob/main/datasets/javascript-libs-from-top-1mm-sites/charts/most-used-js-libs-2022-06-05.svg) showing that jQuery leads the way! Ken Wheeler's [Slick](https://kenwheeler.github.io/slick/) also does very well. WordPress seems to have a big influence here.
- 📆 [Sails](https://sailsjs.com/) is a MVC-oriented webapp framework for Node and an online Sails event, [Sailsconf](https://sailsconf.com/), is taking place [on YouTube](https://www.youtube.com/channel/UCje9Wo6cbh3IERItPbyq1QA) on June 22-24 (next Weds-Fri).
- The [end-of-life date for Node.js 16](https://nodejs.org/en/blog/announcements/nodejs16-eol/) has been brought forward to September 2023 to coincide with the EOL of OpenSSL 1.1.1.

[ow 1.0: Function Argument Validation for Humans](https://sindresorhus.com/ow/) — A fluent API to define constraints for function arguments (e.g. `ow(input, ow.string.minLength(5))`) and get nice error messages on failure. Now a pure ESM package. [GitHub repo.](https://github.com/sindresorhus/ow)

[zx v7.0: Better Shell Scripting with JavaScript](https://github.com/google/zx/releases/tag/7.0.0) — If writing JavaScript sounds better than wrestling with bash, Perl or Python, `zx` is for you. v7.0 is rewritten in TypeScript, has an [`echo`](https://github.com/google/zx#echo) function for printing the output of other scripts, [`within`](https://github.com/google/zx#within) for creating new async contexts, and a new interactive REPL mode (behind `-i`).
