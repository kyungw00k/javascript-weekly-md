[Developer Implements 'Doom' in TypeScript's Type System](https://socket.dev/blog/typescript-types-running-doom) — A curious quirk of TypeScript’s type system is that [it is Turing-complete](https://github.com/microsoft/TypeScript/issues/14833) which has led some developers to implement apps entirely *in* the type system. One such developer has spent eighteen months producing 177 *terabytes* of types to get 1993’s Doom running with them. Ridiculous and amazing in equal measure, he [▶️ explains the project in this widely lauded 7-minute video.](https://www.youtube.com/watch?v=0mCsluv5FXA) 👏

Sarah Gooding (Socket)
