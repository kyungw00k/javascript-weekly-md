# JavaScript Weekly Issue 509: October 9, 2020

## Top Stories

⚙️ So many Rust-powered JavaScript tools came across our path this week that there's definitely a bit of a theme in this issue! Both Rust and Go seem particularly suited to the JS tools space, so it'll be neat to see further developments.

![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60/t3o0a88sfap1i7mbdwvk.jpg)

[ZzFXM: A Tiny JavaScript Music Generator](https://keithclark.co.uk/articles/zzfxm/) — Built for ‘size-limited productions’, ZzFX lets you create a wide array of sounds with very little code. Enjoy [some examples.](https://keithclark.github.io/ZzFXM/#examples) Even more impressively there’s a [tracker system](https://keithclark.github.io/ZzFXM/tracker/?url=https://raw.githubusercontent.com/keithclark/ZzFXM/master/examples/songs/sanxion.js) for creating songs for it.

Keith Clark and Frank Force

[RSLint: A Super Fast JS Linter Written in Rust](https://github.com/RDambrosio016/RSLint) — Last week we mentioned the Go-powered [esbuild](https://github.com/evanw/esbuild) bundler but now Rust wants a turn at creating performance-focused JavaScript tooling. This seems to be a bit of a trend as Dr. Axel mentions next..

Riccardo D'Ambrosio

[Writing JavaScript Tools in Other Languages – A New Trend?](https://2ality.com/2020/10/js-plus-other-languages.html) — Given what we just said about the item above, we think there’s something to Dr. Axel’s observation and are happy to see more of it.

Dr. Axel Rauschmayer

![](https://copm.s3.amazonaws.com/9d5c42c7.png)

[16 JavaScript Debugging Tips You Probably Didn’t Know](https://raygun.com/learn/javascript-debugging-tips?utm_medium=newsletter&utm_source=javascriptweekly&utm_campaign=cooperpress&utm_content=article) — Debug your JavaScript with greater speed and accuracy using these lesser-known tips and tricks.

Raygun sponsor

[Volta: Install, Run, and Manage JavaScript Tools Quickly](https://github.com/volta-cli/volta) — *“No matter the package manager, Node runtime, or OS, one command is all you need:* `volta install`*.”* It’s a bold claim, but this tool provides a way both to manage tools like `node`, `npm` and `yarn` but [also switch versions](https://docs.volta.sh/guide/understanding) when projects require it.

Volta

**⚡️ Quick bytes:**

- React user? The React core team wants you to [take its React 2020 community survey](https://www.surveymonkey.co.uk/r/FRFQWPH) so they can make the documentation better suit your needs.
- The ESLint project [reflects upon a year of paying contributors.](https://eslint.org/blog/2020/10/year-paying-contributors-review) It seems to have worked well for them.
- Chrome 87 is bringing [camera pan, tilt and zoom](https://web.dev/camera-pan-tilt-zoom/) controls to the Web via a JavaScript API.
- Some [minification benchmarks](https://github.com/privatenumber/minification-benchmarks) comparing babel-minify, esbuild, terser and UglifyJS.
- Want to chat with other Angular developers? There's now an [official Angular Discord server.](https://discord.com/invite/angular)
- The [latest VS Code release](https://code.visualstudio.com/updates/v1_50) improves JS debugging with real-time view of performance metrics and improved stepping.
- Next.js user? There's [a Next.js newsletter](https://nextjsnews.com/) now. It's not ours, but it looks pretty good.

## 📚 Tutorials, Opinions and Stories

[The File System Access API: Simplifying Access to Local Files](https://web.dev/file-system-access/) — You can now read files and save changes directly to those same files directly from JavaScript in the browser. It has an [open standard spec](https://wicg.github.io/file-system-access/) but [isn't widely implemented yet](https://caniuse.com/native-filesystem-api) save for [Chrome 86+](https://developers.google.com/web/updates/2020/10/nic86)

Pete LePage and Thomas Steiner

[jQuery to React: How Dropbox Rewrote the HelloSign Editor](https://dropbox.tech/application/jquery-to-react--how-we-rewrote-the-hellosign-editor) — A case study of how Dropbox went about modernizing its HelloSign e-signature acquisition by replacing 12,000 lines of jQuery-based code.

Asa Ayers
