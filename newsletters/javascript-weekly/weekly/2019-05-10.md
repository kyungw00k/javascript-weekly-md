**⏩ IN BRIEF:**

- Microsoft has [launched React Native for Windows](https://techcrunch.com/2019/05/06/microsoft-launches-react-native-for-windows/), bringing the once native mobile development framework to Windows 10 desktop, Xbox, and tablet ecosystems too.
- Googlebot has [gotten an upgrade](https://webmasters.googleblog.com/2019/05/the-new-evergreen-googlebot.html) which means it can now crawl and run all your latest ES6 JavaScript, handle Web Components, and more.
- A DoS vulnerability [has been found in the popular Axios HTTP library](https://snyk.io/blog/a-denial-of-service-vulnerability-discovered-in-the-axios-javascript-package-affecting-all-versions-of-the-popular-http-client/) where a malicious server operator could gum down your client-side code.
- The [latest version of GraalVM](https://medium.com/graalvm/announcing-graalvm-19-4590cf354df8), a JVM-related VM from Oracle, includes a *"JavaScript engine compliant with the latest ECMAScript 2019 standard"* which provides a migration path for anyone using Rhino or Nashorn.
- To celebrate Minecraft's 10th birthday, Mojang have rebuilt [Minecraft Classic in JavaScript](https://classic.minecraft.net/) and made it free to [play](https://classic.minecraft.net/). The [source](https://classic.minecraft.net/assets/js/app.js) shows they've used [Babylon.js](https://www.babylonjs.com/) for the 3D.
